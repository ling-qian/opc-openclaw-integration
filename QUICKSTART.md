# OPC Platform - 快速启动指南

## 🚀 5分钟快速上手

### 第一步：访问平台

**平台地址**: https://opc-platform.onrender.com

> ⚠️ 注意：Render免费计划在15分钟无请求后会休眠，首次访问可能需要等待30秒唤醒。

### 第二步：注册账号

1. 点击右上角"注册"按钮
2. 填写用户名、邮箱、密码
3. 点击"创建账户"
4. 自动登录并跳转到平台

### 第三步：完善资料

1. 点击右上角头像，选择"个人资料"
2. 填写个人信息：
   - 显示名称
   - 个人简介
   - 所在地
   - 个人网站
3. 添加技能标签：
   - 输入技能名称，按Enter添加
   - 例如：Python、AI、设计、写作
4. 点击"保存修改"

### 第四步：浏览项目

1. 点击导航栏"揭榜挂帅"
2. 浏览67个AI项目
3. 使用筛选功能：
   - 按行业筛选
   - 按预算筛选
   - 按关键词搜索
4. 查看项目详情

### 第五步：智能匹配

1. 系统会根据你的技能自动推荐项目
2. 点击"推荐项目"查看匹配结果
3. 查看匹配分数和匹配原因
4. 选择感兴趣的项目

### 第六步：提交投标

1. 在项目详情页点击"提交投标"
2. 填写投标信息：
   - 项目方案
   - 报价金额
   - 预计工期
3. 提交投标

### 第七步：其他功能

#### 学习成长
- 点击"学院"查看精品课程
- 匹配合适的导师
- 参与学习社区

#### 健康管理
- 点击"健康"记录运动
- 查看排行榜
- 获得成就徽章

#### 超级个体
- 点击"超级个体"体验AI Agent
- 创建数字分身
- 与AI对话

## 🔧 常见问题

### Q: 忘记密码怎么办？
A: 目前暂不支持找回密码，请联系管理员重置。

### Q: 如何修改用户名？
A: 用户名创建后不可修改，请谨慎选择。

### Q: 投标后多久能收到回复？
A: 通常3-7个工作日，具体取决于项目方。

### Q: 如何提高匹配分数？
A: 完善技能标签，添加具体的技术栈和行业经验。

### Q: 支持哪些支付方式？
A: 目前支持支付宝、微信支付、银行转账。

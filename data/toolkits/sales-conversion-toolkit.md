## 一、销售文案模板

### 模板1：问题-方案型

```
标题：[目标人群] 每天都在犯的 [N] 个错误

你是不是也这样？
- 错误1：______（导致什么后果）
- 错误2：______（导致什么后果）
- 错误3：______（导致什么后果）

这些错误让你 [具体损失]。

我之前也一样，后来发现关键在于 [核心洞察]。

现在我用 [方法]，[具体结果]。

想了解完整方法？
→ [行动指引]
```

### 模板2：故事型

```
标题：从 [起点状态] 到 [终点状态]，我只做了这一件事

[起点故事：你的困境，50-100字]

[转折：发现了什么，50字]

[行动：做了什么，100字]

[结果：具体改变，50字]

如果你也想 [目标]，
→ [行动指引]
```

### 模板3：对比型

```
标题：[A方法] vs [B方法]，差距在哪？

大多数人用 [A方法]：
- 问题1
- 问题2
- 问题3

少数人用 [B方法]：
- 优势1
- 优势2
- 优势3

核心区别在于 [关键差异]。

想掌握 [B方法]？
→ [行动指引]
```

### 模板4：紧迫感型

```
标题：[限时/限量] [具体优惠]

[痛点描述]

[解决方案介绍]

[价值证明：案例/数据]

⚠️ [紧迫感：仅限前X名/截止X日]

→ [立即行动]
```

### 模板5：社会证明型

```
标题：已经有 [数字] 人通过 [方法] 实现了 [结果]

[客户证言1]
[客户证言2]
[客户证言3]

[服务/产品介绍]

→ [行动指引]
```

## 二、成交漏斗设计模板

### 基础漏斗

```
[公域曝光] → 小红书/抖音/知乎内容
    ↓ 3-5% 关注
[关注] → 公众号/账号粉丝
    ↓ 10-20% 加群
[入群] → 微信群/社群
    ↓ 5-10% 付费
[成交] → 低价引流品
    ↓ 10-20% 升级
[复购] → 中高价产品
    ↓ 20-30% 推荐
[裂变] → 口碑传播
```

### 高级漏斗（带自动化）

```
[广告投放] → 落地页
    ↓ 自动
[注册] → 收集联系方式
    ↓ 自动邮件序列

    ↓ 自动
[成交] → 限时优惠
    ↓ 人工跟进
[交付] → 服务/产品
    ↓ 自动
[复购] → 续费/升级
```

## 三、定价策略参考

### 定价心理学

```
锚定效应：先展示高价，再展示中价
尾数定价：¥299 比 ¥300 感觉便宜很多
套餐效应：组合比单买感觉更值
稀缺效应：限时/限量提升购买意愿
```

### 定价公式

```
成本定价：成本 × 2-3倍 = 售价
价值定价：帮客户节省/赚取的价值 × 10-20%
竞品定价：参考竞品，高10-30%（差异化）
感知定价：根据目标客户的支付意愿
```

### 定价阶梯

| 产品层级 | 价格区间 | 目的 |
|----------|----------|------|
| 免费 | ¥0 | 曝光+信任 |
| 引流品 | ¥9-99 | 筛选付费用户 |
| 核心品 | ¥99-999 | 核心利润 |
| VIP品 | ¥999-2999 | 高利润+深度服务 |
| 定制品 | ¥3000+ | 超级用户 |

## 四、私信话术

### 话术1：新关注者

```
嗨，感谢关注！
看到你对 [领域] 感兴趣，我最近在做 [服务/内容]，
专门帮 [人群] 解决 [问题]。
有什么问题随时找我聊 👋
```

### 话术2：内容互动者

```
嗨，看到你 [点赞/评论/收藏] 了我的内容，
谢谢支持！
这篇内容只是冰山一角，完整方法我在 [服务/课程] 里详细讲了。
有兴趣了解吗？
```

### 话术3：主动咨询者

```
你好，看到你对 [服务] 感兴趣，
简单介绍一下：
[服务内容，2-3句话]
[核心价值，1句话]
[价格和名额]
要了解一下详细信息吗？
```

### 话术4：犹豫者

```
理解你的顾虑，[顾虑点] 确实需要考虑。
不过 [解决顾虑的论点]。
我之前有个学员也有同样的担心，后来 [成功案例]。
要不要先看看 [免费内容/案例] 了解一下？
```

### 话术5：成交后

```
感谢信任！🎉
接下来我会 [交付流程说明]。
有任何问题随时找我。
我们第一次 [服务时间] 开始，期待一起 [目标]！
```

### 话术6：跟进未回复

```
嗨，之前给你发的消息不知道看到没？
最近比较忙的话完全理解。
如果之后有需要，随时找我就好 👋
```

### 话术7：老客户复购

```
嗨 [客户名]，
上次的 [服务/产品] 用得怎么样？
最近我推出了 [新产品/升级服务]，
基于上次的经验，我觉得特别适合你。
有兴趣了解一下吗？
```

### 话术8：请求推荐

```
嗨 [客户名]，
很高兴之前的 [服务] 对你有帮助！
如果你身边有朋友也有类似需求，
欢迎推荐给他们。
推荐成功的话，[激励措施]。
```

### 话术9：处理异议-价格

```
理解价格是重要的考虑因素。
不过换个角度想，[价值换算]。
而且 [超预期的价值]。
要不要先试试 [低门槛选项]？
```

### 话术10：处理异议-时间

```
理解时间紧张。
其实 [服务] 正是帮你 [节省时间/提升效率] 的。
每周只需要 [时间投入]，[具体收益]。
很多学员反馈做了之后反而 [时间收益]。
```

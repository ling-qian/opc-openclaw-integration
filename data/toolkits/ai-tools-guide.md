## 六、AI工具使用技巧

### 1. Prompt工程

```
❌ "写一篇文章"（太模糊）
✅ "你是一个小红书博主，擅长[领域]，请写一篇关于[主题]的笔记，
   300字以内，包含3个要点，风格轻松活泼，结尾引导收藏"（具体）
```

### 2. 角色设定

```
"你是一个有10年经验的[领域]专家，请从专业角度分析[问题]"
"你是一个小红书爆款写手，请帮我优化这篇内容的标题"
"你是一个销售专家，请帮我写一段关于[产品]的销售话术"
```

### 3. 迭代优化

```
第一轮：生成初稿
第二轮："请让这段内容更口语化/更专业/更简短"
第三轮："请加入一个真实案例"
第四轮："请优化结尾的行动号召"
```

## 二、短内容模板

### 模板1：痛点拆解型

```
你是不是也遇到过 [具体问题]？

原因通常是这3个：
1. ______
2. ______
3. ______

解决方法：______

试试看，有用的话记得收藏 ⭐
```

### 模板2：干货分享型

```
一个 [领域] 的小技巧：

[具体方法]

效果：[具体结果]

我用了 [时间]，[真实反馈]

关注我，每天分享一个 [领域] 干货 👆
```

### 模板3：故事引入型

```
[朋友/我] 之前 [遇到的问题]，试了很多方法都没用。

后来发现关键在于 [核心洞察]，只做了 [简单动作]，[具体结果]。

完整方法我放在下一条笔记，想看的扣 1 👇
```

### 模板4：对比分析型

```
[A方法] vs [B方法]，差距在哪？

大多数人用 [A方法]：
- 问题1
- 问题2

少数人用 [B方法]：
- 优势1
- 优势2

核心区别：______

你更常用哪个？
```

### 模板5：清单型

```
[领域] 的 [N] 个技巧，建议收藏：

1. ______
2. ______
3. ______
4. ______
5. ______

最后一个最实用！你觉得呢？
```

## 三、长内容框架

### 框架1：问题-方案型

```
标题：[目标人群] 每天都在犯的 [N] 个错误

开头：痛点共鸣

正文：
- 错误1 + 正确做法
- 错误2 + 正确做法
- 错误3 + 正确做法

结尾：总结 + CTA
```

### 框架2：故事型

```
标题：从 [起点] 到 [终点]，我只做了这一件事

开头：你的困境

正文：
- 转折点
- 具体行动
- 遇到的困难
- 如何克服

结尾：成果 + 启示 + CTA
```

### 框架3：指南型

```
标题：[主题] 完整指南 [年份] 版

开头：为什么写这篇

正文：
- 第一步：______
- 第二步：______
- 第三步：______
- 常见问题FAQ

结尾：总结 + 推荐资源 + CTA
```

### 框架4：案例分析型

```
标题：拆解 [案例] 如何 [实现目标]

开头：案例介绍

正文：
- 背景
- 策略
- 执行
- 结果
- 可借鉴的点

结尾：总结 + 你的应用 + CTA
```

### 框架5：观点型

```
标题：[反常识观点]

开头：抛出观点

正文：
- 为什么大多数人认为 [旧观点]
- 事实是什么
- 证据/案例
- 正确的理解

结尾：总结 + 互动（你同意吗？）
```

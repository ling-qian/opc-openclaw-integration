# 课程03：销售转化体系课

## Sales Convert System

> "The purpose of a business is to create a customer." — Peter Drucker

---

## 🎯 课程目标

建立系统化的销售转化能力，让内容自动带来收入。

**学完这门课，你将能够：**
- 写出有转化力的销售文案
- 设计从曝光到成交的完整漏斗
- 设计并推出你的第一款付费产品
- 在社群中建立信任并实现转化
- 搭建24/7自动运转的销售系统

---

## 📋 课程信息

| 项目 | 详情 |
|------|------|
| **适合人群** | 有流量但变现困难、不知道怎么卖东西的创作者 |
| **课时** | 6个模块，30天体系搭建 |
| **每日投入** | 1-1.5小时 |
| **前置要求** | 已有一定的内容产出基础 |
| **核心工具** | 文案模板 + 漏斗工具 + 自动化平台 |

|------|------|----------|

## 🗓️ 30天学习计划

| 天数 | 学习内容 | 产出目标 |
|------|----------|----------|
| Day 1-5 | SM1 + SM2 | 掌握销售心理学 + 写出3篇销售文案 |
| Day 6-12 | SM3 | 设计完成你的成交漏斗 |
| Day 13-18 | SM4 | 设计并发布第一款付费产品 |
| Day 19-25 | SM5 | 在社群中完成首次销售 |
| Day 26-30 | SM6 + 总复盘 | 搭建自动化销售系统 |

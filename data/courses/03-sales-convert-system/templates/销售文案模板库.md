# 销售文案模板库

> 5种经过验证的销售文案模板，覆盖从引流到成交的全链路。直接填写即可使用。

---

## 模板1：痛点切入型文案

**适用场景**：朋友圈、小红书、社群推广
**核心逻辑**：痛点→方案→结果→行动

```
【标题】：________________________

【痛点描述（2-3句）】
你是不是也这样？
[描述用户的具体痛苦场景]...

【共鸣加深（1-2句）】
其实[目标人群]中，90%的人都有这个问题。
不是你不够努力，而是[根本原因]。

【解决方案（2-3句）】
我花了[时间]，总结了一套[方法名称]：
✅ [好处1]
✅ [好处2]
✅ [好处3]

【结果展示（1-2句）】
已经帮[数量]人[具体成果]。
比如[客户案例简述]。

【行动号召（1-2句）】
[限时/限量信息]
[行动指令：点击链接/私信我/扫码]

【价格信息】
原价[原价]，限时[优惠价]
[零风险保证]
```

---

## 模板2：故事型文案

**适用场景**：公众号长文、小红书图文、详情页
**核心逻辑**：故事→转折→方法→证明→行动

```
【开头：引入故事（100字）】
[时间]前，我遇到了一个[问题]。
当时我[描述困境的状态]...

【发展：尝试与失败（150字）】
我试过[方法1]，没用。
也试过[方法2]，还是不行。
甚至花了[金额]找[外部帮助]，结果[不满意的结果]。

【转折：发现方法（100字）】
直到有一天，我[发现了什么]。
我才意识到，原来[核心洞察]。

【高潮：方法介绍（200字）】
这套方法叫[名称]，核心是：
1. [步骤1]：[说明]
2. [步骤2]：[说明]
3. [步骤3]：[说明]

【结果：效果证明（100字）】
用了这个方法后：
📊 [数据结果1]
📊 [数据结果2]
⭐ [客户评价]

【结尾：行动号召（50字）】
现在，我把这套方法做成了[产品形式]。
[价格信息]
[行动指令]
[零风险保证]
```

---

## 模板3：对比型文案

**适用场景**：产品对比、服务介绍、课程推广
**核心逻辑**：现状→理想→差距→方案→行动

```
【标题】：从[现状]到[理想]，你只差这一步

【现状描述（50字）】
现在的你：
❌ [痛点1]
❌ [痛点2]
❌ [痛点3]

【理想状态（50字）】
理想中的你：
✅ [好处1]
✅ [好处2]
✅ [好处3]

【差距分析（50字）】
差距在哪？
不是[常见误区1]，也不是[常见误区2]。
而是你缺少[核心解决方案]。

【方案介绍（100字）】
[产品名称]能帮你：
1. [具体价值1]
2. [具体价值2]
3. [具体价值3]

包含：
📦 [内容1]
📦 [内容2]
📦 [内容3]

【价格与行动（50字】】
价格：[原价] → [优惠价]
保证：[零风险承诺]
行动：[立即行动指令]
```

## 模板4：见证型文案

**适用场景**：客户案例推广、口碑营销
**核心逻辑**：案例→方法→结果→引出产品

```
【标题】：[客户称呼]用[时间]做到了[成果]

【客户背景（50字）】
[客户称呼]，[职业/身份]。
[他/她]之前[描述之前的困境]。

【使用过程（100字）】
[客户称呼]找到我时，[描述当时的问题]。
我给[他/她]的建议是：
1. [建议1]
2. [建议2]
3. [建议3]

[客户称呼]按照这个方法执行了[时间]。

【成果展示（50字）】
结果：
📊 [数据成果1]
📊 [数据成果2]
💬 客户原话："[真实评价]"

【引出产品（50字）】
这不是个例。
我已经帮[数量]人实现了类似的成果。

现在，我把这套方法做成了[产品名称]。
[价格信息]
[行动指令]
```

## 模板5：紧迫型文案

**适用场景**：限时促销、活动推广、招生倒计时
**核心逻辑**：价值→稀缺→紧迫→行动

```
【标题】：⚠️ [紧急通知/最后XX小时/仅剩XX名额]

【价值重申（50字）】
[产品名称]，帮你[核心价值]。
包含：[核心内容清单]
价值：[总价值金额]

【优惠信息（50字）】
现在报名/购买：
💰 原价[原价]，今天只要[优惠价]
🎁 额外赠送[赠品]（价值[金额]）

【稀缺性（30字）】
⚠️ 本期仅限[名额]人
目前已报名[已报名人数]人
仅剩[剩余名额]个名额

【紧迫性（30字）】
⏰ 优惠截止时间：[具体时间]
倒计时：[时间]

【行动指令（20字）】
👇 立即[行动]
[链接/二维码/联系方式]

【零风险保证（30字）】
✅ [保证1]
✅ [保证2]
不满意？[退款政策]
```

## 使用指南

| 模板 | 最佳场景 | 转化率 | 使用频率 |
|------|---------|--------|---------|
| 痛点切入 | 日常推广 | ⭐⭐⭐⭐ | 每周2-3次 |
| 故事型 | 长内容推广 | ⭐⭐⭐⭐⭐ | 每周1次 |
| 对比型 | 产品介绍 | ⭐⭐⭐⭐ | 按需使用 |
| 见证型 | 口碑营销 | ⭐⭐⭐⭐⭐ | 每周1-2次 |
| 紧迫型 | 限时促销 | ⭐⭐⭐⭐⭐ | 活动期间 |

**组合使用建议**：
1. 平时用「痛点切入」+「故事型」建立信任
2. 推广时用「对比型」+「见证型」展示价值
3. 促销时用「紧迫型」促成成交

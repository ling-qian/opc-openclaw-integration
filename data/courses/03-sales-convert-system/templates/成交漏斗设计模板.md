# 成交漏斗设计模板

> 一套完整的成交漏斗设计框架，从流量入口到成交转化，帮你系统化地设计销售流程。

---

## 一、漏斗总览

```
[流量入口] → [内容吸引] → [建立信任] → [激发需求] → [促成成交] → [售后复购]
   ↓              ↓             ↓            ↓            ↓            ↓
 曝光量         浏览量         关注量        咨询量        成交量        复购量
 10000    →    3000     →     500    →     100    →      20     →      5
```

### 环节3：建立信任

**信任建设路径**：

| 阶段 | 用户行为 | 你的动作 | 目标 |
|------|---------|---------|------|
| 初次接触 | 看到内容 | 持续输出优质内容 | 建立第一印象 |
| 二次接触 | 关注账号 | 发送欢迎消息 | 建立连接 |
| 三次接触 | 看更多内容 | 展示专业度 | 建立信任 |
| 四次接触 | 私信/评论 | 提供免费价值 | 深化关系 |
| 五次接触 | 咨询服务 | 展示案例和评价 | 促成转化 |

**信任素材清单**：
- [ ] 客户案例（至少3个）
- [ ] 客户评价截图（至少5条）
- [ ] 个人专业背景介绍
- [ ] 服务过程展示
- [ ] 免费价值内容（PDF/模板/工具）

### 环节4：激发需求

**需求激发话术模板**：

| 场景 | 话术 |
|------|------|
| 用户咨询时 | "你目前遇到的最大问题是？" |
| 用户犹豫时 | "如果不解决这个问题，3个月后会怎样？" |
| 用户对比时 | "和自己摸索相比，用我的方法可以节省____时间" |
| 用户问价时 | "这个价格包含____，相当于每天____元" |

**常见异议处理**：

| 异议 | 回应策略 |
|------|---------|
| "太贵了" | 拆解到每天成本+对比替代方案成本 |
| "没时间" | 说明时间投入+效率提升 |
| "不确定有没有用" | 展示案例+提供保证 |
| "再考虑考虑" | 给限时优惠+制造紧迫感 |

### 环节5：促成成交

**成交触发器**：

| 触发器 | 具体做法 | 示例 |
|--------|---------|------|
| 限时优惠 | 设置截止时间 | "前100名立减100元" |
| 限量稀缺 | 限制名额 | "本期仅限20人" |
| 零风险保证 | 退款承诺 | "7天无理由退款" |
| 赠品加码 | 额外赠送 | "今天下单送XX" |
| 社会证明 | 展示成交 | "已有XX人购买" |

**成交话术模板**：

```
基于你刚才说的情况，我建议你选择[方案名称]。

这个方案能帮你解决[核心问题]，包含：
1. [价值点1]
2. [价值点2]
3. [价值点3]

原价[原价]，今天[优惠条件]可以享受[优惠价]。

而且有[保证承诺]，完全没有风险。

你觉得怎么样？要现在安排吗？
```

### 环节6：售后复购

| 阶段 | 时间 | 动作 | 目标 |
|------|------|------|------|
| 成交后 | 立即 | 发送感谢+交付物 | 确认购买决策 |
| 使用中 | 第3天 | 询问使用情况 | 及时解决问题 |
| 使用后 | 第7天 | 请求评价 | 获取口碑素材 |
| 复购期 | 第30天 | 推荐升级产品 | 促进复购 |

## 四、优化检查清单

| 检查项 | 状态 | 优化方向 |
|--------|------|---------|
| 流量是否足够？ | □ | 增加渠道/加大投入 |
| 内容是否吸引人？ | □ | 优化标题/封面/开头 |
| 信任是否建立？ | □ | 增加案例/评价/背景介绍 |
| 需求是否激发？ | □ | 强化痛点/展示结果 |
| 成交是否顺畅？ | □ | 简化流程/增加保证 |
| 复购是否跟进？ | □ | 建立售后SOP |

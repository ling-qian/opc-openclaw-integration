## 三、跟进策略

### 跟进频率建议

| 客户阶段 | 跟进频率 | 跟进方式 | 跟进内容 |
|---------|---------|---------|---------|
| 潜在 | 每周1次 | 内容互动 | 点赞/评论/私信 |
| 意向 | 每3天1次 | 私信 | 发送相关案例/资料 |
| 咨询 | 每天1次 | 私信/电话 | 解答疑问/推进流程 |
| 报价 | 每2天1次 | 私信 | 确认意向/处理异议 |
| 成交 | 每周1次 | 私信 | 使用跟进/满意度调查 |
| 复购 | 每月1次 | 私信 | 新品推荐/专属优惠 |

### 跟进话术模板

**首次接触**：
```
你好！我是[你的名字]，做[你的领域]的。
看到你[来源/互动]，想问下你目前有没有[相关需求]？
```

**跟进提醒**：
```
上次聊的[话题]，你考虑得怎么样了？
有什么问题可以随时问我~
```

**成交推进**：
```
基于你的情况，我推荐[方案名称]。
包含[核心内容]，能帮你[核心价值]。
现在报名还有[优惠信息]，你觉得怎么样？
```

**售后跟进**：
```
你好！你用[产品/服务]有一段时间了，
使用体验怎么样？有没有遇到什么问题？
```

## 七、客户标签系统

### 标签分类

| 标签类型 | 标签选项 |
|---------|---------|
| 来源渠道 | 小红书/抖音/朋友圈/转介绍/其他 |
| 客户阶段 | 潜在/意向/咨询/报价/成交/复购/流失 |
| 需求类型 | 需求A/需求B/需求C |
| 预算等级 | 低(<500)/中(500-2000)/高(>2000) |
| 活跃度 | 高活跃/中活跃/低活跃/沉默 |
| 满意度 | 非常满意/满意/一般/不满意 |

### 标签使用说明

- 每个客户可以有多个标签
- 标签随沟通进展更新
- 用标签筛选客户群，精准营销

# SM1: 销售心理学

> 为什么人们买东西

---

## 🎯 模块目标

**学完本模块，你将能够：**
- 说出人们购买的6个核心动机
- 识别并消除购买过程中的阻力
- 用心理学原理提升转化率
- 建立以用户为中心的销售思维

## 📖 核心知识点

### 1. 购买的本质：解决问题

人们买东西不是因为"想要"，而是因为"需要"。

```
痛点 → 需求 → 购买 → 解决问题
```

**核心原则：** 卖的不是产品，是问题的解决方案。

### 2. 购买的6个核心动机

| 动机 | 说明 | 例子 |
|------|------|------|
| **恐惧** | 害怕失去、害怕落后 | 「再不学AI就要被淘汰了」 |
| **贪婪** | 想要更多、想要更好 | 「用这个方法，收入翻倍」 |
| **虚荣** | 想被认可、想与众不同 | 「成为朋友圈最懂XX的人」 |
| **懒惰** | 想省事、想走捷径 | 「3天掌握XX，不用苦练」 |
| **好奇** | 想知道、想了解 | 「这个方法99%的人不知道」 |
| **从众** | 别人都在用、不想掉队 | 「10万人已经学会了」 |

### 3. 购买阻力的4个来源

即使有需求，人们也会犹豫。原因：

| 阻力 | 说明 | 解决方法 |
|------|------|----------|
| **不信任** | 不相信你说的是真的 | 用案例、数据、背书建立信任 |
| **不紧急** | 觉得以后再买也行 | 制造紧迫感、限时优惠 |
| **不值得** | 觉得价格太高 | 强调价值、提供保障 |
| **不确定** | 不确定是否适合自己 | 提供试用、降低门槛 |

### 4. 信任公式

```
信任 = 专业度 × 可靠度 × 亲密度 ÷ 自私度
```

- **专业度**：你懂不懂这个领域？
- **可靠度**：你说的话能不能兑现？
- **亲密度**：用户觉得你亲近吗？
- **自私度**：你是不是只想赚他的钱？

**降低自私度的关键：** 先提供价值，再提销售请求。

### 5. 销售漏斗的心理学

```
认知 → 兴趣 → 信任 → 购买 → 复购 → 推荐
```

每个阶段，用户的心理状态不同：

| 阶段 | 心理状态 | 你需要做的 |
|------|----------|------------|
| 认知 | 「这是什么？」 | 吸引注意力 |
| 兴趣 | 「好像有用」 | 提供价值 |
| 信任 | 「这个人靠谱」 | 建立信任 |
| 购买 | 「我想买」 | 促成交易 |
| 复购 | 「还想买」 | 超预期交付 |
| 推荐 | 「推荐给朋友」 | 制造口碑 |

### 方法2：销售话术框架

**FABE法则：**
```
F - Feature（特点）：这个产品是什么
A - Advantage（优势）：它比别的好在哪
B - Benefit（好处）：对你有什么用
E - Evidence（证据）：凭什么相信我
```

**例子：**
```

B：让你系统掌握一人公司的方法论
E：已经有500+人学完并实现了首单收入
```

## ⚠️ 常见误区

### 误区1：「好产品不用卖」

**真相：** 再好的产品，如果用户不知道、不理解、不信任，也卖不出去。销售是让好产品到达需要它的人。

### 误区2：「销售就是忽悠」

**真相：** 好的销售是帮用户做出正确的决定。如果你的产品真的能解决问题，销售就是帮他们迈出这一步。

### 误区3：「降价就能卖出去」

**真相：** 降价不是解决销售问题的方法。如果用户不信任你，免费都不要。先解决信任问题。

### 误区4：「我不需要懂心理学」

**真相：** 销售的本质是理解人。懂心理学，你就懂用户在想什么，就能更好地帮他们做出决定。

### 误区5：「销售话术太套路了」

**真相：** 话术是工具，关键是你怎么用。真诚 + 专业 + 话术 = 最佳组合。

# SM2: 文案写作

> 销售文案的底层公式

---

## 🎯 模块目标

掌握销售文案的核心公式，写出有转化力的文案。

**学完本模块，你将能够：**
- 写出吸引点击的标题
- 写出让人想买的正文
- 写出促进行动的CTA
- 完成一篇完整的销售文案

---

## 📖 核心知识点

### 1. 销售文案的核心公式

**AIDA公式（最经典）：**
```
A - Attention：吸引注意力
I - Interest：引发兴趣
D - Desire：激发欲望
A - Action：促进行动
```

**PAS公式（适合痛点型）：**
```
P - Problem：描述问题
A - Agitate：放大痛点
S - Solution：给出方案
```

**4P公式（适合产品型）**
```
P - Promise：做出承诺
P - Picture：描绘画面
P - Proof：提供证据
P - Push：推动行动
```

### 2. 标题的5种写法

| 类型 | 公式 | 例子 |
|------|------|------|
| **数字型** | 数字 + 结果 | 「3个方法让你的转化率翻倍」 |
| **痛点型** | 痛点 + 方案 | 「写了100篇文章还没变现？因为缺这一步」 |
| **故事型** | 人物 + 转折 | 「从月入0到月入5万，我只做了这一件事」 |
| **反常识型** | 颠覆认知 | 「为什么我不建议你先做内容」 |
| **疑问型** | 提出问题 | 「你的副业为什么一直做不起来？」 |

### 3. 正文的写作技巧

**技巧1：AIDA结构**
```
【Attention】
用一个震撼的事实/数据/问题开头

【Interest】
描述用户的具体痛点和场景

【Desire】
展示解决方案和好处
提供案例和证据

【Action】
给出明确的行动指令
制造紧迫感
```

**技巧2：使用"你"而不是"我"**

❌ 我的课程有很多干货
✅ 你将学到10个实战方法

**技巧3：具体化**

❌ 这个方法很有效
✅ 这个方法让我的转化率从2%提升到8%

### 4. CTA的写法

**CTA = 告诉用户下一步做什么**

**好的CTA：**
- 「点击下方链接，立即加入」
- 「现在报名，立减200元」
- 「扫码添加微信，领取免费资料」

**差的CTA：**
- 「欢迎购买」
- 「可以考虑一下」
- 「感兴趣的话了解一下」

### 5. 文案的信任元素

| 元素 | 作用 | 例子 |
|------|------|------|
| **数据** | 用数字证明 | 「帮助500+学员实现首单」 |
| **案例** | 用故事证明 | 「学员小王，3个月收入翻倍」 |
| **评价** | 用口碑证明 | 「这是我学过最实用的课程」 |
| **保障** | 降低风险 | 「7天无理由退款」 |
| **权威** | 用身份证明 | 「10年营销经验」 |

## 🛠️ 实操方法

### 方法1：文案创作流程

```
Step 1：明确目标
- 这篇文案要达成什么？（关注/购买/加微信）
- 目标用户是谁？

Step 2：收集素材
- 用户痛点是什么？
- 产品卖点是什么？
- 有哪些案例/数据？

Step 3：写标题
- 用5种公式各写1个标题
- 选最好的1个

Step 4：写正文
- 用AIDA结构
- 加入信任元素

Step 5：写CTA
- 明确的行动指令
- 制造紧迫感

Step 6：修改润色
- 删掉废话
- 检查逻辑
- 测试阅读体验
```

### 方法2：文案拆解法

找一篇你觉得写得好的销售文案，拆解它的结构：

```
标题：______
吸引注意力的方式：______

正文结构：
- 开头：______
- 痛点：______
- 方案：______
- 证据：______
- CTA：______

你觉得好的地方：______
你可以借鉴的地方：______
```

- 具体说明
- 案例/数据

### 好处3

- 案例/数据

## ⚠️ 常见误区

### 误区1：「文案要写得很长」

**真相：** 文案的长度取决于产品的复杂度和价格。低价产品，短文案就够了。

### 误区2：「文案要写得很文艺」

**真相：** 销售文案的核心是转化，不是文学。清晰、直接、有说服力 > 华丽辞藻。

### 误区3：「我不需要学文案」

**真相：** 一人公司的每一篇内容，都是文案。学会写文案，就是学会赚钱。

### 误区4：「模板太死板了」

**真相：** 模板是起点，不是终点。先用模板写出第一篇，再慢慢优化和个性化。

### 误区5：「写完就不用改了」

**真相：** 好文案是改出来的。写完后至少修改3遍，每遍都会更好。

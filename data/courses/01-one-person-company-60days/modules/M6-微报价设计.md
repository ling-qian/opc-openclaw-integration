# M6: 微报价设计 — 你的第一款产品

> 不需要复杂产品，用最轻的方式测试市场、收取第一笔钱

---

## 6.1 什么是微报价

**微报价 = 最小可行的付费服务**

- 不需要开发完整产品
- 不需要建课程体系
- 不需要技术工具
- 只需要：你 + 客户 + 解决问题的能力

**本质：** 用1对1服务验证市场需求，收集真实反馈。

## 6.2 为什么微报价是最佳起步方式

| 优势 | 说明 |
|------|------|
| 零成本 | 不需要任何前期投入 |
| 快速验证 | 1周内就能知道市场要不要 |
| 收集反馈 | 直接听到客户真实需求 |
| 建立案例 | 成功案例是最好的内容素材 |
| 积累信任 | 1对1服务建立深度关系 |

## 6.3 最优模板：$1000 / 4次服务

**服务设计：**
- 4次每周1对1视频/语音通话
- 每次 45-60分钟
- 定价：$1000（约 ¥7000）

### 为什么是 $1000？

**逻辑一：验证价值**
- 有人愿意付 $1000 → 你的服务有价值
- 后续可推出 $3000-$10,000 的升级服务
- 愿意付 $1000 的客户，对高价敏感度低

**逻辑二：足够深度**
- 4次通话足够帮客户拿到实际结果
- 能建立深度信任
- 收集到真实的反馈和案例

**逻辑三：可规模化**
- 1对1服务覆盖人数有限
- 将流程整理为课程，实现规模化

## 6.4 服务设计三步法

### 第一步：锁定核心痛点

从三大永恒赛道中选择：
- **健康**：减肥、健身、养生、心理
- **财富**：副业、投资、理财、职场
- **关系**：恋爱、婚姻、社交、亲子

**痛点越具体越好：**
```
❌ "帮人提升效率"（太泛）
✅ "帮996打工人每天省出2小时"（具体）
```

### 第二步：描绘理想状态

```
现状：[客户现在的困境]
理想：[客户想要到达的状态]
差距：[需要跨越的鸿沟]
```

**示例：**
```
现状：每天加班到10点，回家只想躺平
理想：每天6点下班，有时间学习和健身
差距：不会用AI工具，工作效率低
```

### 第三步：设计服务流程

```
第1次：诊断问题，制定计划
第2次：教授方法，实操练习
第3次：复盘优化，解决卡点
第4次：总结成果，下一步规划
```

### 服务命名

好名字的特征：
- 包含时间承诺（60天）
- 包含具体结果（省2小时）
- 有吸引力（零成本、爆改）

**命名公式：**
```
[时间] + [具体结果] + [计划/挑战/训练营]

例：
"60天效率爆改计划"
"30天零成本副业启动挑战"
"21天写作变现训练营"
```

## 6.5 微报价推广策略

### 内容推广

在每篇长内容末尾加入：
```
如果你也想 [目标]，我提供 [服务名称]，
4次1对1帮你 [具体结果]。

名额有限，私信「报名」了解详情 👇
```

### 私信转化

当用户点赞、评论、收藏你的内容时：
1. 感谢关注
2. 简要介绍你的服务
3. 邀请了解详情

### 问卷收集

如果暂时没有人私信，可以在内容中引导：
```
我正在准备一期 [服务名称]，
想了解你的需求，填一个1分钟问卷：
[链接]
```

## 6.6 服务升级路径

```
微报价 ($1000)
  → 积累案例和反馈
小班课 ($299/人 × 10人)
  → 标准化流程
录播课程 ($99)
  → 规模化
社群会员 ($99/月)
  → 持续复利
```

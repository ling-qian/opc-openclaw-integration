# M4: 写作变现 — 核心技能一

> 写作是一切内容的基础，掌握它就掌握了流量密码

---

## 4.1 为什么写作是第一技能

所有社交媒体内容的核心都是写作：
- 短视频 → 先写脚本
- 小红书 → 先写文案
- 公众号 → 直接写作
- 直播 → 先写话术
- 播客 → 先写大纲

**写作 = 用文字解决问题的能力**，不需要文笔好。

## 4.2 短内容：快速测试，精准获客

### 定义
- 字数：≤ 300字
- 时长：≤ 1分钟（视频类）
- 频率：日更 1-3 条

### 核心作用
1. 测试创意（哪个方向有反馈）
2. 吸引精准受众
3. 维持账号活跃度
4. 高互动内容可扩展为长内容

### 短内容模板

**模板一：痛点拆解**
```
你是不是也遇到过 [具体问题]？

原因通常是这3个：
1. ______
2. ______
3. ______

解决方法：______

试试看，有用的话记得收藏 ⭐
```

**模板二：干货分享**
```
一个 [领域] 的小技巧：

[具体方法]

效果：[具体结果]

我用了 [时间]，[真实反馈]

关注我，每天分享一个 [领域] 干货 👆
```

**模板三：故事引入**
```
[朋友/我] 之前 [遇到的问题]，试了很多方法都没用。

后来发现关键在于 [核心洞察]，只做了 [简单动作]，[具体结果]。

完整方法我放在下一条笔记，想看的扣 1 👇
```

## 4.3 长内容：建立信任，核心变现载体

### 定义
- 字数：1000-3000字
- 频率：每周 1-2 篇
- 平台：公众号、知乎、小红书长笔记

### 核心作用
1. 建立专业信任
2. 延长用户停留时间
3. 完整阐述解决方案
4. 引导用户进入变现漏斗

### 长内容框架

```
标题：引发好奇 + 承诺价值

开头：痛点共鸣 / 故事引入 / 反常识观点

正文：
  - 问题拆解（为什么会这样）
  - 解决方案（具体怎么做）
  - 案例佐证（真实效果）
  - 注意事项（避坑指南）

结尾：
  - 总结核心要点
  - 行动指引（下一步做什么）
  - 引导互动（评论/收藏/私信）
```

## 4.4 创意验证流程

```
想法 → 短内容测试（看互动数据）
         ↓
    数据好 → 长内容扩展
         ↓
    效果好 → 视频呈现
         ↓
    反馈好 → 免费服务试错
         ↓
    成功 → 付费产品落地
```

**关键原则：** 不要一次性做重，先用最小成本验证。

## 4.5 内容创作4大方向

### 方向一：痛点拆解
拆解赛道常见痛点，给出具体可落地的解决方案
```
"上班族每天浪费2小时的3个习惯"
```

### 方向二：改写优化
分析同领域高互动内容，用自身视角改写
```
看到爆款 → 分析为什么火 → 加入自己的经验 → 改写发布
```

### 方向三：学习笔记
读书、听播客时，输出有钩子、有主体、有结论的笔记
```
"读完《原子习惯》，我用1个方法戒掉了熬夜"
```

### 方向四：标题延伸
借鉴热门标题模式，延伸撰写详细解读
```
原标题："5个让你变穷的习惯"
延伸："5个让你变穷的习惯，第3个90%的人都在犯"
```

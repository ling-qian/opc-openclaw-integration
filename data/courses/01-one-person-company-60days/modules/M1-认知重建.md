# M1: 认知重建 — 一人公司是什么

> 打破误区，建立正确的创业认知基础

---

## 1.1 一人公司 ≠ 孤独创业

**常见误区：** "一人公司就是一个人啥都干"

**真相：** 一人公司是一种 **商业形态**，不是一种 **人数限制**。

核心特征：
- **轻资产**：无需租办公室、招员工
- **高灵活**：随时调整方向，试错成本极低
- **强复利**：内容、技能、人脉持续积累
- **可外包**：设计、技术、客服都可以找人做，你只把控核心

## 1.2 2026年为什么是一人公司元年

三大趋势汇聚：

1. **AI 工具爆发** — 一个人能干过去一个团队的活
   - 写作：AI 辅助生成初稿、改写、翻译
   - 设计：Canva + AI 出图，无需专业设计
   - 视频：剪映 + AI 字幕、配音
   - 客服：AI 自动回复常见问题

2. **平台基础设施成熟** — 变现路径已打通
   - 公众号付费文章
   - 小红书商品笔记
   - 知乎盐选专栏
   - 微信小商店 / 小鹅通

3. **远程工作常态化** — 用户接受线上服务
   - 1对1 视频咨询已成主流
   - 线上课程市场持续增长
   - 社群付费意愿提升

## 1.3 轻资产创业的底层逻辑

```
传统创业：资金 → 产品 → 营销 → 用户 → 收入
一人公司：内容 → 用户 → 反馈 → 产品 → 收入
```

**关键区别：**
- 传统创业先投入大量资金，再找用户（风险高）
- 一人公司先用内容吸引用户，再根据需求造产品（风险低）

**你的投入：** 时间 + 注意力 + 执行力（不需要钱）

## 1.4 心态准备

### 你会经历的阶段

```
兴奋期（第1-7天）
  → "这也太简单了吧！"
  ↓
迷茫期（第8-21天）
  → "怎么没人看？我是不是不适合？"
  ↓
低谷期（第22-40天）
  → "要不算了？回去上班也挺好的"
  ↓
突破期（第41-60天）
  → "原来如此！第一条爆款来了！"
  ↓
复利期（第60天+）
  → "停不下来了，内容自己跑"
```

### 应对策略

- **迷茫时**：回到对标账号，分析他们前100条内容
- **低谷时**：只做最低限度——每天发1条短内容，不断更
- **想放弃时**：记住 Dan Koe 的故事——发了80条视频才迎来第一条爆款

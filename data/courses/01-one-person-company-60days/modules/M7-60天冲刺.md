## 7.1 60天总览

```
Day 1-3    调研定位（M2+M3）
```

## 7.3 Day 4-60：日更执行

### 每日最低标准

```
短内容：3条/天（最低1条）
长内容：1篇/周（最低2周1篇）
私信：5条/天
互动：30分钟/天（回复评论、评论他人内容）
```

### 短内容发布节奏

| 时间 | 内容类型 |
|------|----------|
| 早8点 | 干货分享 / 技巧 |
| 中12点 | 痛点拆解 / 问题 |
| 晚8点 | 故事 / 案例 / 互动 |

### 灵感来源

1. **对标账号** — 改写他们的高互动内容
2. **评论区** — 用户的问题就是选题
3. **搜索框** — 平台搜索联想词
4. **日常记录** — 随时记灵感到灵感库
5. **书籍/播客** — 学习笔记输出

## 7.4 引流策略

### 核心原则

> 算法不会带来一夜爆红，社交增长的核心是主动发力。

### 每日30分钟引流动作

| 动作 | 时间 | 说明 |
|------|------|------|
| 评论他人内容 | 10分钟 | 同赛道大号+小号，有深度的评论 |
| 私信互动 | 10分钟 | 感谢关注、建立连接 |
| @相关博主 | 5分钟 | 长内容中@相关博主，增加曝光 |
| 回复评论 | 5分钟 | 及时回复自己内容下的评论 |

### 私信策略

每周30条私信，60天累计240条。

**私信对象：**
- 点赞/收藏你内容的人
- 评论区提问的人
- 关注你的新粉丝
- 同赛道的创作者

**私信模板：**
```
嗨，看到你关注了 [话题]，
我最近在做 [服务/内容]，专门帮 [人群] 解决 [问题]。

如果你也有类似困扰，可以看看这篇 [内容链接]，
有问题随时找我聊 👋
```

**关键指标：** 0.5% 回复率 = 240条 × 0.5% = 1.2单 ≈ 1单 $1000

## 7.5 微报价转化

### 时间线

```
Day 7    发布微报价服务介绍
Day 7-14 在内容中持续推广
Day 14+  开始私信转化
Day 21+  跟进意向客户
Day 30+  目标：成交首单
```

### 转化流程

```
内容曝光 → 用户互动 → 私信沟通 → 了解需求 → 介绍服务 → 成交
```

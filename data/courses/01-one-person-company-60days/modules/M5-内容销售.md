# M5: 内容销售 — 核心技能二

> 销售 = 讲故事 + 解决问题，不是硬推销

---

## 5.1 重新理解销售

**错误认知：** 销售 = 推销、说服、求人买
**正确认知：** 销售 = 帮用户看清问题，提供解决方案

一人公司的销售，90%通过内容完成，无需面对面沟通。

## 5.2 三步销售公式

```
痛点呈现 → 理想结果描绘 → 独特落地路径
```

### 第一步：痛点呈现

让用户意识到"我确实有这个问题"：

```
❌ "你需要提升效率"（太抽象）
✅ "你每天是不是花2小时在无效会议上？开完会啥也没记住？"（具体可感知）
```

### 第二步：理想结果描绘

让用户看到"解决后会怎样"：

```
❌ "你会变得更高效"（太空洞）
✅ "每天省出2小时，一个月多出60小时，够你学一门新技能"（具体可想象）
```

### 第三步：独特路径

提供一条"只有你能给"的解决路径：

```
❌ "早起、列清单、番茄钟"（烂大街）
✅ "我用AI工具自动处理80%的重复工作，每天只专注4小时核心任务"（独特）
```

## 5.3 销售文案模板

### 模板一：问题-方案型

```
标题：[目标人群] 每天都在犯的 [N] 个错误

你是不是也这样？
- 错误1：______
- 错误2：______
- 错误3：______

这些错误让你 [具体损失]。

我之前也一样，后来发现关键在于 [核心洞察]。

现在我用 [方法]，[具体结果]。

想了解完整方法？[行动指引]
```

### 模板二：故事型

```
标题：从 [起点状态] 到 [终点状态]，我只做了这一件事

[起点故事：你的困境]

[转折：发现了什么]

[行动：做了什么]

[结果：具体改变]

如果你也想 [目标]，[行动指引]
```

### 模板三：对比型

```
标题：[A方法] vs [B方法]，差距在哪？

大多数人用 [A方法]：
- 问题1
- 问题2

少数人用 [B方法]：
- 优势1
- 优势2

核心区别在于 [关键差异]。

想掌握 [B方法]？[行动指引]
```

## 5.4 从内容到成交的漏斗

```
曝光（内容被看到）
  ↓
兴趣（用户停留阅读）
  ↓
信任（觉得你专业靠谱）
  ↓
需求（意识到自己需要帮助）
  ↓
行动（私信/填问卷/购买）
```

**每个环节都可以通过内容优化：**

| 环节 | 内容策略 |
|------|----------|
| 曝光 | 热点话题、关键词优化、高频发布 |
| 兴趣 | 痛点标题、故事开头、反常识观点 |
| 信任 | 真实案例、数据佐证、持续输出 |
| 需求 | 痛点放大、理想描绘、紧迫感 |
| 行动 | 明确CTA、低门槛入口、限时优惠 |

## 5.5 常见销售误区

### 误区一：等产品做好再卖
**真相：** 先卖再做，用需求驱动产品

### 误区二：免费内容太多，没人付费
**真相：** 免费内容建立信任，付费内容提供深度/个性化

### 误区三：定价太低
**真相：** 低价 = 低价值感知，合理定价反而更容易成交

### 误区四：不好意思推销
**真相：** 你在帮人解决问题，不是在求人

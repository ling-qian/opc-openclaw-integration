# M3: 搭建主场 — 自媒体主页体系

> 选对平台，优化主页，让陌生人 3 秒内知道你能帮什么

---

## 3.1 平台选择：不同平台的特性

| 平台 | 内容形式 | 适合场景 | 流量逻辑 |
|------|----------|----------|----------|
| **公众号** | 长文 | 深度内容、信任建立 | 社交裂变 + 搜一搜 |
| **小红书** | 图文/短视频 | 种草、测试创意 | 推荐算法 + 搜索 |
| **知乎** | 问答/长文 | 专业背书、SEO 长尾 | 搜索 + 推荐 |
| **抖音** | 短视频 | 快速曝光、引流 | 推荐算法 |
| **B站** | 中长视频 | 深度教程、粉丝粘性 | 推荐 + 搜索 |

### 推荐组合策略

**起步期（第1-30天）：**
- 主阵地：**小红书**（测试创意快、反馈快）
- 辅助：**公众号**（沉淀长内容）

**成长期（第30-60天）：**
- 主阵地：**公众号 + 小红书**
- 加入：**知乎**（长尾流量）

**成熟期（60天+）：**
- 全平台分发，根据数据调整重心

## 3.2 主页优化：3秒法则

用户看到你的主页，3秒内必须知道：

1. **你是谁** — 一句话定位
2. **你能帮什么** — 核心价值
3. **为什么要关注你** — 差异化/信任背书

### 昵称公式

```
❌ "小明的日常"（看不出价值）
✅ "小明 | 职场效率教练"（定位清晰）
✅ "小明教副业"（直接说明价值）
```

### 简介公式

```
[身份定位] + [核心价值] + [行动指引]

例：
"10年产品经理 | 帮职场人用 AI 工具每天省2小时
私信「效率」领取免费模板"
```

### 头像/封面

- 专业感 > 生活感
- 真人头像 > 风景/卡通（建立信任）
- 封面图包含核心价值主张

## 3.3 对标账号分析法

找10个同赛道账号，逐一分析：

### 分析维度

| 维度 | 看什么 |
|------|--------|
| 定位 | 一句话怎么介绍自己？ |
| 内容 | 发什么类型的内容？哪些互动高？ |
| 频率 | 多久发一次？ |
| 爆款 | 哪些内容点赞/收藏最多？为什么？ |
| 变现 | 卖什么产品/服务？定价多少？ |
| 不足 | 哪里做得不够好？你能怎么改进？ |

### 重点：找不足

对标账号的不足 = 你的机会：
- 案例不接地气 → 你用国内真实案例
- 方法太复杂 → 你简化为3步
- 不够细分 → 你专注特定人群

## 3.4 内容基础设施

### 灵感库

建立一个随时记录灵感的系统：

```markdown
# 灵感库

## 来自小红书
- [ ] 看到一篇关于XXX的笔记，可以写：______

## 来自日常
- [ ] 今天遇到XXX问题，可以写解决方案

## 来自对标账号
- [ ] XXX的爆款可以这样改写：______
```

### 内容日历

| 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 |
|------|------|------|------|------|------|------|
| 短内容×3 | 短内容×3 | 短内容×3 | 短内容×3 | 短内容×3 | 长内容×1 | 复盘 |

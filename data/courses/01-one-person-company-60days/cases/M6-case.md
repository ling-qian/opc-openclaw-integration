# M6 实战案例：微报价从设计到成交第一单的完整过程

## 人物背景

**刘洋**，27岁，设计师，选择"Logo设计"作为一人公司方向。目标：设计一个微报价，在7天内成交第一单。

## 微报价设计过程

### Step 1：确定服务产品

刘洋最初的想法是"什么都做"：

> Logo设计、海报设计、名片设计、UI设计、包装设计...

通过课程学习，他意识到**微报价的核心是"小而精"**，于是聚焦到：

| 项目 | 原计划 | 微报价方案 |
|------|--------|-----------|
| 服务范围 | 所有设计 | 仅Logo设计 |
| 交付周期 | 7-14天 | 48小时 |
| 价格 | 面议 | 299元/个 |
| 修改次数 | 不限 | 3次 |
| 附加服务 | 无 | 送名片设计 |

### Step 2：微报价画布填写

用课程提供的「微报价设计画布」：

| 要素 | 内容 |
|------|------|
| **一句话描述** | 48小时交付你的品牌Logo，299元起 |
| **目标客户** | 刚创业的小老板、个体户、微商 |
| **核心痛点** | 找设计公司太贵（2000+），自己做太丑 |
| **解决方案** | 专业设计师48小时出图，价格只要299 |
| **交付物** | 3个Logo方案 + 源文件 + 名片设计 |
| **保证承诺** | 不满意全额退款 |
| **价格** | 299元（原价599，限时5折） |

### Step 3：销售渠道搭建

刘洋选择了3个渠道同时测试：

**渠道1：闲鱼**
- 标题：`【48小时出图】Logo设计299元 送名片设计 不满意退款`
- 配图：过往作品展示（9张）
- 价格设置：299元

**渠道2：小红书**
- 内容：发布Logo设计过程的短视频
- 引流方式：评论区引导私信
- 话术："想做Logo的宝子可以私信我，48小时出图~"

**渠道3：朋友圈**
- 内容：每天发1条设计相关的内容
- 包括：设计过程、客户反馈、行业知识

### Step 4：成交第一单

**时间线**：

| 时间 | 事件 | 详情 |
|------|------|------|
| Day 1 | 发布闲鱼 | 浏览量120，咨询0 |
| Day 2 | 发布小红书 | 浏览量85，咨询1（未成交） |
| Day 3 | 闲鱼优化标题 | 浏览量提升到280 |
| Day 4 | 收到闲鱼咨询 | 客户问"能做餐饮Logo吗？" |
| Day 4 | 回复+报价 | 详细沟通需求，确认299元 |
| Day 5 | 开始设计 | 3个方案初稿完成 |
| Day 5 | 提交客户 | 客户选了方案B，提出2处修改 |
| Day 6 | 修改完成 | 客户确认满意 |
| Day 6 | 交付+收款 | 闲鱼确认收货，299元到账 |

**第一单客户画像**：
- 年龄：35岁
- 职业：开了一家小型奶茶店
- 需求：需要一个简单的Logo用于招牌和外卖平台
- 选择原因：价格合适+48小时出图+有退款保证

### Step 5：成交话术复盘

闲鱼上的关键对话：

```
客户：能做餐饮Logo吗？
刘洋：可以的~我做过很多餐饮类的Logo，您是做什么品类的？
客户：奶茶店
刘洋：好的，我先了解一下您的需求：
1. 店名叫什么？
2. 有没有喜欢的风格？
3. 有没有必须包含的元素？
我出3个方案给您选，48小时内交付，不满意全额退款~
客户：价格多少？
刘洋：299元，包含3个方案+源文件+名片设计
客户：行，我先发你需求
```

**关键点**：
1. 先问需求，再报价——显得专业
2. 强调"3个方案"——降低选择风险
3. "不满意退款"——消除客户顾虑

## 关键经验

1. **微报价要"小到让人无法拒绝"**——299元+48小时+退款保证，几乎零风险
2. **闲鱼是最好的冷启动渠道**——有明确的购买意图，转化率比小红书高5倍
3. **先做再优化**——第一版报价不完美没关系，根据客户反馈调整
4. **48小时交付是核心竞争力**——大公司要7-14天，你只要2天，这就是差异化

# 🚀 一人公司 60 天启动实战课

> 从零到 $1000 的完整路径 | 基于 Dan Koe 方法论 + 国内落地场景

**适合人群：** 想创业但无启动资金、想做副业但不知从何入手的普通人

**课程目标：** 60天内建立一人公司，斩获第一笔 $1000（约 ¥7000）收入

**学习方式：** 边学边做，每周有明确任务和交付物

---

## 详细课程内容

### [M1: 认知重建](modules/M1-认知重建.md)
- 一人公司 ≠ 孤独创业
- 2026年为什么是一人公司元年
- 轻资产创业的底层逻辑
- 心态准备：煎熬、怀疑与突破

### [M2: 找准赛道](modules/M2-找准赛道.md)
- 从搜索记录中发现你的赛道
- 从购买行为中验证市场需求
- "市场饱和"恰恰是机会
- 实操：输出你的赛道定位文档

### [M3: 搭建主场](modules/M3-搭建主场.md)
- 平台选择：公众号 vs 小红书 vs 知乎 vs 抖音
- 主页优化：让陌生人 3 秒内知道你能帮什么
- 对标账号分析法
- 实操：完成账号搭建 + 10个对标分析

### [M4: 写作变现](modules/M4-写作变现.md)
- 短内容：≤300字，日更1-3条
- 长内容：每周1-2篇深度文
- 创意验证流程
- 内容创作4大方向
- 实操：产出30条短内容 + 2篇长内容

### [M5: 内容销售](modules/M5-内容销售.md)
- 销售 = 讲故事，不是硬推销
- 三步公式：痛点→理想结果→路径
- 90%销售靠内容完成
- 实操：撰写3套销售文案

### [M6: 微报价设计](modules/M6-微报价设计.md)
- 什么是微报价？为什么是最佳起步方式
- $1000 定价的底层逻辑
- 4次1对1服务流程设计
- 服务命名与包装
- 实操：输出完整微报价方案

### [M7: 60天冲刺](modules/M7-60天冲刺.md)
- Day 1-3：调研定位
- Day 4-60：日更执行
- 引流策略：每日30分钟主动出击
- 私信转化：每周30条，60天成交首单
- 实操：完成60天计划表

### [M8: 规模化](modules/M8-规模化.md)
- 从1对1到课程化
- 社群运营与会员体系
- 个人品牌IP打造
- 路径图：$1000 → $10,000 → $100,000

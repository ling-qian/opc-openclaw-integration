# 剪映视频制作指南

> 从拍摄到发布的完整流程，零基础也能做出专业短视频。

---

## 一、前期准备

### 设备清单

| 设备 | 推荐 | 预算 | 必要性 |
|------|------|------|--------|
| 手机 | iPhone 12+/华为P40+ | 已有 | ⭐⭐⭐⭐⭐ |
| 三脚架 | 手机桌面三脚架 | 30-80元 | ⭐⭐⭐⭐⭐ |
| 麦克风 | 领夹麦/指向麦 | 50-200元 | ⭐⭐⭐⭐ |
| 灯光 | 环形灯/柔光灯 | 50-300元 | ⭐⭐⭐ |
| 背景 | 干净墙面/书架 | 0元 | ⭐⭐⭐⭐ |

### 拍摄设置

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| 分辨率 | 1080P | 4K太大，720P太模糊 |
| 帧率 | 30fps | 日常足够 |
| 比例 | 9:16（竖屏） | 小红书/抖音/快手 |
| 对焦 | 人脸追踪 | 保持清晰 |
| 曝光 | 手动锁定 | 避免忽明忽暗 |

## 二、拍摄技巧

### 口播型视频

**机位设置**：
```
      [灯光]
        ↓
   ┌─────────┐
   │  手机   │ ← 三脚架，镜头与眼睛平齐
   │  📱    │
   └─────────┘
        ↕ 50-80cm
      [你] 🧑
```

**拍摄要点**：
- 看镜头，不看屏幕
- 语速适中，不要太快
- 表情自然，适当加手势
- 每段话之间停顿1秒（方便剪辑）

### 教程型视频

**屏幕录制设置**：
1. 打开剪映 → 录屏
2. 设置录制区域
3. 开始操作演示
4. 边操作边讲解

**拍摄要点**：
- 操作要慢，让观众看清
- 关键步骤放大显示
- 用箭头/高亮标注重点

### 剧情型视频

**分镜拍摄**：
1. 先写好脚本（分场景）
2. 每个场景单独拍摄
3. 多角度拍摄（正面/侧面/特写）
4. 每个镜头拍2-3遍

## 三、剪映剪辑流程

### Step 1：导入素材

1. 打开剪映App
2. 点击"开始创作"
3. 选择要剪辑的视频/图片
4. 点击"添加"

### Step 2：基础剪辑

| 操作 | 方法 | 用途 |
|------|------|------|
| 分割 | 点击视频 → 分割 | 剪掉多余部分 |
| 删除 | 选中片段 → 删除 | 去掉废片 |
| 调整顺序 | 长按拖动 | 调整片段顺序 |
| 变速 | 点击片段 → 变速 | 加速/减速 |
| 裁剪 | 点击片段 → 裁剪 | 调整画面范围 |

### Step 3：添加文字

**字幕添加**：
1. 点击底部"文字"
2. 选择"新建文本"
3. 输入字幕内容
4. 调整位置、大小、颜色

**自动字幕**：
1. 点击"文字" → "识别字幕"
2. 自动识别语音生成字幕
3. 检查并修改错误

**字幕样式推荐**：
| 位置 | 样式 | 适用 |
|------|------|------|
| 底部居中 | 白色黑边 | 通用 |
| 顶部居中 | 黄色加粗 | 强调 |
| 侧边 | 小字 | 补充说明 |

### Step 4：添加音乐

1. 点击底部"音频" → "音乐"
2. 搜索合适的音乐
3. 调整音量（建议背景音乐20-30%）
4. 裁剪音乐长度

**音乐推荐**：
| 视频类型 | 音乐风格 |
|---------|---------|
| 干货教程 | 轻快商务 |
| 故事分享 | 温暖抒情 |
| 剧情视频 | 紧张悬疑 |
| 产品展示 | 时尚潮流 |

### Step 5：添加特效

**常用特效**：
| 特效 | 用途 | 添加方式 |
|------|------|---------|
| 转场 | 场景切换 | 两段视频之间 |
| 滤镜 | 统一色调 | 底部"滤镜" |
| 贴纸 | 增加趣味 | 底部"贴纸" |
| 动画 | 文字效果 | 文字 → 动画 |
| 画中画 | 多画面 | 底部"画中画" |

### Step 6：导出发布

1. 点击右上角"导出"
2. 选择分辨率：1080P
3. 选择帧率：30fps
4. 等待导出完成
5. 保存到相册

## 四、剪辑模板

### 60秒口播视频剪辑流程

```
0-3秒：封面/钩子画面 + 大字标题
3-15秒：第一段内容 + 字幕
15-30秒：第二段内容 + 字幕
30-45秒：第三段内容 + 字幕
45-55秒：总结/CTA + 字幕
55-60秒：结尾画面 + 引导关注
```

### 教程视频剪辑流程

```
0-10秒：成果展示（先给观众看效果）
10-20秒：准备阶段（需要什么）
20-80秒：操作演示（分步骤）
80-100秒：成果对比
100-120秒：总结 + CTA
```

## 五、效率技巧

### 批量制作

1. 做好一个视频模板
2. 保存为"草稿模板"
3. 每次只替换素材和文字
4. 批量导出

### 常用快捷操作

| 操作 | 方法 |
|------|------|
| 撤销 | 双指点击屏幕 |
| 重做 | 三指点击屏幕 |
| 放大时间线 | 双指张开 |
| 缩小时间线 | 双指捏合 |
| 复制片段 | 长按 → 复制 |

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 视频模糊 | 拍摄时对焦清晰，导出选1080P |
| 音画不同步 | 重新对齐音频和视频 |
| 字幕错别字 | 用自动识别后手动校对 |
| 导出很慢 | 关闭其他App，清理缓存 |
| 文件太大 | 降低分辨率或压缩 |

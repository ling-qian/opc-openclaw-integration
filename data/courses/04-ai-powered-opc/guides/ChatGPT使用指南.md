# ChatGPT 使用指南

> 详细的 prompt 技巧 + 10个实战场景，让你用 ChatGPT 效率提升10倍。

---

## 一、Prompt 基础技巧

### 万能 Prompt 公式

```
角色 + 任务 + 要求 + 格式

示例：
你是一个小红书运营专家（角色），
请帮我写一篇关于Excel技巧的图文笔记（任务），
要求标题吸引人、内容口语化、300字以内（要求），
输出格式：标题 + 正文 + 5个话题标签（格式）。
```

### 6个核心技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| 指定角色 | 让AI扮演专家 | "你是一个有10年经验的PPT设计师" |
| 明确任务 | 说清楚要做什么 | "请帮我写/分析/优化/生成..." |
| 给出上下文 | 提供背景信息 | "我的目标用户是25-35岁的职场白领" |
| 限定格式 | 规定输出形式 | "用表格/列表/代码块输出" |
| 设置约束 | 明确限制条件 | "字数300字以内/不要用专业术语" |
| 给出示例 | 提供参考样本 | "类似这样的风格：[示例]" |

### Prompt 优化对比

| 原始 Prompt | 优化后 Prompt |
|-------------|--------------|
| 帮我写个标题 | 你是一个小红书运营专家，请为"Excel快捷键"这个主题写5个标题，要求：包含数字、有情绪词、20字以内 |
| 帮我分析一下 | 你是一个数据分析专家，请分析以下销售数据，找出3个关键趋势，用表格格式输出，每个趋势附上具体数字 |
| 帮我写篇文章 | 你是一个公众号写手，请写一篇1500字的文章，主题是"如何用Excel提高工作效率"，目标读者是职场新人，风格轻松幽默，包含3个具体案例 |

## 二、10个实战场景

### 场景1：选题生成

```
Prompt：
你是一个[赛道]领域的资深内容创作者。
请帮我生成20个小红书选题，要求：
1. 覆盖4种类型：干货、故事、资源、互动
2. 每种类型5个
3. 标题要吸引人，包含数字或情绪词
4. 面向[目标用户]
5. 输出格式：编号 + 标题 + 类型
```

### 场景2：标题优化

```
Prompt：
请帮我优化以下5个小红书标题，每个给出3个版本：
1. 数字型（包含具体数字）
2. 痛点型（戳中用户痛点）
3. 悬念型（制造好奇心）

原标题：
1. [标题1]
2. [标题2]
3. [标题3]

输出格式：原标题 → 版本A / 版本B / 版本C
```

### 场景3：内容扩写

```
Prompt：
请根据以下大纲，写一篇500字的小红书图文：

主题：[主题]
大纲：
1. [要点1]
2. [要点2]
3. [要点3]

要求：
- 语言口语化，像朋友聊天
- 每个要点配一个具体案例
- 结尾有互动引导
- 附5个话题标签
```

### 场景4：视频脚本

```
Prompt：
请写一个60秒的短视频脚本：
主题：[主题]
类型：口播型
平台：抖音/小红书

要求：
1. 前3秒有强钩子（制造好奇/抛出痛点）
2. 正文分3个要点，每个15秒
3. 结尾有CTA（关注/评论/私信）
4. 标注：画面描述 + 台词 + 字幕
```

### 场景5：销售文案

```
Prompt：
请为我的[产品名称]写一篇销售文案：
产品：[产品描述]
价格：[价格]
目标客户：[客户画像]
核心卖点：[卖点1]、[卖点2]、[卖点3]

要求：
1. 标题吸引人
2. 先戳痛点，再给方案
3. 包含社会证明（案例/数据）
4. 有明确的CTA
5. 300字以内
```

### 场景6：客户话术

```
Prompt：
我是一个[职业]，提供[服务]。
客户经常会问以下问题，请帮我准备回答话术：
1. "太贵了"
2. "不确定有没有用"
3. "我再考虑考虑"
4. "有没有更便宜的方案"
5. "多久能看到效果"

要求：
- 每个问题给出2个版本的回答
- 版本A：直接回答
- 版本B：反问引导
- 语气要专业但不生硬
```

### 场景7：数据分析

```
Prompt：
以下是我最近30天的内容数据，请帮我分析：

[粘贴数据表格]

请分析：
1. 哪类内容表现最好？为什么？
2. 哪类内容表现最差？为什么？
3. 有什么规律或趋势？
4. 下一步应该怎么优化？
5. 给出3个具体的改进建议

输出格式：结论 + 数据支撑 + 建议
```

### 场景8：竞品分析

```
Prompt：
请帮我分析以下3个竞品账号：

竞品A：[账号名]，粉丝[X]，主要内容[描述]
竞品B：[账号名]，粉丝[X]，主要内容[描述]
竞品C：[账号名]，粉丝[X]，主要内容[描述]

分析维度：
1. 内容策略（类型/频率/风格）
2. 变现方式
3. 优势和劣势
4. 我能学习什么
5. 我的差异化机会在哪里
```

### 场景9：社群运营

```
Prompt：
我有一个[领域]的付费社群，[人数]人。
请帮我设计一周的社群运营计划：

要求：
1. 每天1个主题
2. 包含互动话题、干货分享、案例讨论
3. 有具体的时间安排
4. 有引导话术
5. 目标：提升活跃度和满意度
```

### 场景10：产品设计

```
Prompt：
我想设计一个[产品类型]，请帮我梳理：

基本信息：
- 我的技能：[技能]
- 目标用户：[用户画像]
- 预期价格：[价格区间]

请帮我：
1. 设计产品结构（包含哪些模块）
2. 列出每个模块的核心内容
3. 设计交付流程
4. 制定定价策略
5. 规划推广渠道

输出格式：用表格和列表
```

## 三、高级技巧

### Chain of Thought（链式思考）

```
Prompt：
请一步一步分析这个问题：
[问题]

步骤1：先分析现状
步骤2：再找出原因
步骤3：然后提出方案
步骤4：最后评估可行性

每一步都要给出具体的分析和结论。
```

### Few-Shot（少样本学习）

```
Prompt：
请按照以下范例的风格，写一篇类似的内容：

范例：
[粘贴一个优秀范例]

现在请按照同样的风格，写一篇关于[你的主题]的内容。
要求：结构、语气、长度都和范例类似。
```

### 迭代优化

```
Prompt：
你刚才生成的内容很好，但我想做以下调整：
1. [调整点1]
2. [调整点2]
3. [调整点3]

请基于原内容进行修改，保持好的部分，优化需要改进的部分。
```

## 四、避坑指南

| 常见错误 | 正确做法 |
|---------|---------|
| Prompt太短太模糊 | 越具体越好，给足上下文 |
| 一次问太多问题 | 分步骤，一次解决一个问题 |
| 不检查直接用 | AI会犯错，一定要人工审核 |
| 不提供范例 | 给范例能让输出更符合预期 |
| 不迭代优化 | 第一版很少完美，要反复修改 |

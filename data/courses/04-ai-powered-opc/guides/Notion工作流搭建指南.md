## 三、核心模块搭建

### 模块1：任务管理

**创建步骤**：
1. 新建页面 → 命名"任务管理"
2. 选择"看板"视图
3. 设置分组：待办 / 进行中 / 已完成

**数据库字段**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 任务名称 | 标题 | 任务描述 |
| 状态 | 单选 | 待办/进行中/已完成 |
| 优先级 | 单选 | 高/中/低 |
| 截止日期 | 日期 | 完成时间 |
| 分类 | 单选 | 内容/客户/产品/其他 |
| 备注 | 文本 | 补充说明 |

**每日使用流程**：
```
早上：
1. 打开任务管理
2. 查看今日待办
3. 拖拽任务到"进行中"

晚上：
1. 完成的任务拖到"已完成"
2. 添加明天的新任务
3. 更新本周计划
```

### 模块2：内容管理

**选题库**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 选题 | 标题 | 内容标题 |
| 类型 | 单选 | 干货/故事/资源/互动 |
| 平台 | 单选 | 小红书/抖音/公众号 |
| 状态 | 单选 | 待写/写作中/已发布 |
| 优先级 | 单选 | 高/中/低 |
| 发布日期 | 日期 | 计划发布时间 |
| 链接 | 链接 | 发布后的链接 |

**内容日历**：
- 使用"日历"视图
- 每天一个内容卡片
- 拖拽调整发布日期

### 模块3：客户管理

**客户列表**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 客户名称 | 标题 | 昵称/姓名 |
| 来源 | 单选 | 小红书/抖音/转介绍 |
| 阶段 | 单选 | 潜在/咨询/成交/复购 |
| 联系方式 | 文本 | 微信/手机 |
| 需求 | 文本 | 客户需求描述 |
| 跟进日期 | 日期 | 下次跟进时间 |
| 成交金额 | 数字 | 成交金额 |
| 备注 | 文本 | 其他信息 |

**跟进记录**：
- 关联客户列表
- 记录每次沟通内容
- 设置跟进提醒

### 模块4：财务管理

**收入记录**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 收入日期 |
| 金额 | 数字 | 收入金额 |
| 来源 | 单选 | 咨询/课程/模板/带货 |
| 客户 | 关联 | 关联客户表 |
| 备注 | 文本 | 说明 |

**月度报表公式**：
```
本月总收入 = sum(收入记录.金额)
本月咨询收入 = sum(filter(收入记录.来源 == "咨询").金额)
本月课程收入 = sum(filter(收入记录.来源 == "课程").金额)
```

## 五、使用技巧

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| / | 插入任何内容块 |
| /table | 插入表格 |
| /board | 插入看板 |
| /calendar | 插入日历 |
| /toggle | 插入折叠块 |
| @ | 提及页面或日期 |

### 数据库视图

| 视图类型 | 用途 | 示例 |
|---------|------|------|
| 表格 | 查看所有数据 | 客户列表 |
| 看板 | 按状态分组 | 任务管理 |
| 日历 | 按日期排列 | 内容日历 |
| 画廊 | 卡片展示 | 作品集 |
| 时间线 | 甘特图 | 项目进度 |

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 页面太多找不到 | 用收藏功能，标记常用页面 |
| 数据库太复杂 | 先搭建核心字段，后续再扩展 |
| 手机端操作不便 | 复杂操作用电脑端 |
| 数据丢失 | Notion自动保存，不用担心 |
| 想分享给别人 | 设置"共享"权限 |

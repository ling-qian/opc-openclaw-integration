## 📖 核心知识点

### 1. AI写作的核心原则

**AI是助手，不是替代：**
- AI写初稿，你加入观点和经验
- AI提供框架，你填充核心内容
- AI优化表达，你把控方向和质量

### 2. 高效Prompt的写法

**Prompt公式：**
```
角色 + 任务 + 要求 + 格式
```

| 要素 | 说明 | 例子 |
|------|------|------|
| **角色** | 让AI扮演什么角色 | 「你是一个小红书爆款文案专家」 |
| **任务** | 要AI做什么 | 「请帮我写一篇小红书笔记」 |
| **要求** | 具体的要求 | 「300字，口语化，包含3个方法」 |
| **格式** | 输出的格式 | 「标题+正文+标签」 |

### 3. AI写作的5种场景

| 场景 | Prompt思路 | 产出 |
|------|------------|------|
| **选题生成** | 给赛道+受众，要10个选题 | 选题列表 |
| **大纲生成** | 给主题+结构，要大纲 | 文章大纲 |
| **初稿写作** | 给大纲+要求，要全文 | 文章初稿 |
| **润色优化** | 给初稿，要优化 | 优化版本 |
| **多平台适配** | 给原文，要不同平台版本 | 多版本内容 |

### 4. AI写作的常见问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 内容太空洞 | 没给具体要求 | 加入具体场景和限制 |
| 风格不对 | 没指定风格 | 明确说"口语化/专业/幽默" |
| 内容有错误 | AI幻觉 | 人工审核，核实事实 |
| 内容太泛 | 没给具体方向 | 提供更多背景信息 |

### 5. AI写作工作流

```
Step 1：用AI生成5-10个选题
Step 2：筛选最好的1-2个
Step 3：用AI生成大纲
Step 4：确认/调整大纲
Step 5：用AI写初稿
Step 6：加入你的经验和观点
Step 7：用AI润色优化
Step 8：人工最终审核
Step 9：排版发布
```

## 🛠️ 实操方法

### 方法1：选题生成Prompt

```
我是一个做[你的赛道]的创作者，目标受众是[目标人群]。
请帮我生成10个内容选题，要求：
1. 覆盖不同内容类型（干货/故事/观点）
2. 每个选题都能引发目标受众的兴趣
3. 标注适合的平台（小红书/公众号/抖音）
4. 用表格形式输出
```

### 方法2：文章写作Prompt

```
请帮我写一篇[平台]文章，主题是「[主题]」。

背景信息：
- 我的赛道：[你的赛道]
- 目标读者：[目标读者]
- 文章目的：[目的]

要求：
- 字数：[X]字左右
- 结构：[具体结构]
- 风格：口语化，像朋友聊天
- 包含具体案例和方法

请先给出大纲，我确认后再写全文。
```

### 方法3：多平台适配Prompt

```
我有一篇文章，主题是「[主题]」。
请帮我改写成以下平台的版本：
1. 小红书版本（300字，口语化，带标签）
2. 朋友圈版本（100字，真实感，带互动）
3. 知乎版本（500字，专业感，带案例）

原文：[粘贴原文]
```

## ⚠️ 常见误区

### 误区1：「AI写的内容直接发」

**真相：** 必须审核和修改。AI可能有错误，也缺乏你的个人风格。

### 误区2：「Prompt越简单越好」

**真相：** Prompt越详细，产出越好。给AI足够的背景和要求。

### 误区3：「用AI就是作弊」

**真相：** AI是工具，就像用Word写文章一样。关键是你怎么用。

### 误区4：「AI会让我失去写作能力」

**真相：** AI帮你提效，但核心思考和表达还是你的。用AI省下的时间，可以做更多有价值的事。

### 误区5：「一个Prompt就够了」

**真相：** 不同场景需要不同的Prompt。建立你的Prompt库，针对不同需求使用。

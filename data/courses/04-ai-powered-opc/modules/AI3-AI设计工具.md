| 功能 | 说明 | 使用场景 |
|------|------|----------|
| **模板库** | 海量现成模板 | 快速出图 |

| **品牌套件** | 保存品牌颜色/字体 | 保持一致性 |

### 3. 设计的核心原则

**原则1：简洁**
- 不要堆砌元素
- 留白很重要
- 一个页面一个重点

**原则2：对比**
- 大小对比
- 颜色对比
- 粗细对比

**原则3：对齐**
- 元素要对齐
- 保持视觉整齐
- 使用网格辅助

**原则4：重复**
- 保持风格一致
- 使用统一的颜色和字体
- 建立品牌识别

### 5. MidJourney基础

**基本用法：**
```
/imagine [你的描述]
```

**Prompt结构：**
```
[主体] + [风格] + [细节] + [参数]
```

**例子：**
```
A minimalist workspace with laptop, clean and modern style, 
soft lighting, pastel colors --ar 16:9 --v 6
```

## ⚠️ 常见误区

### 误区1：「我不会设计」

**真相：** Canva的模板已经帮你做好了设计，你只需要修改文字和颜色。

### 误区2：「设计要很复杂」

**真相：** 好设计 = 简洁。不要堆砌元素，保持干净清晰。

### 误区3：「我要学Photoshop」

**真相：** 对于一人公司，Canva完全够用。Photoshop太重了。

### 误区4：「AI生成的图片不好看」

**真相：** 关键在于Prompt。写好描述，AI可以生成很高质量的图片。

### 误区5：「设计不重要」

**真相：** 视觉是用户对你的第一印象。好的设计能提升专业感和信任度。

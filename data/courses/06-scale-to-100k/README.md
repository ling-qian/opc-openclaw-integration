# 📈 课程06：规模化增长课 — 从 $1K 到 $100K

> 从"赚到钱"到"赚大钱"的系统化增长路径

**适合人群：** 已经通过一人公司赚到第一笔收入，想要规模化增长的创业者

**课程目标：** 建立从月入 $1,000 到月入 $100,000 的完整增长系统

**学习方式：** 模型驱动，每个模块提供可执行的增长框架

# SC2: 产品矩阵 — 构建从低到高的产品体系

> 一个产品只能赚一次，一套产品能赚一辈子

---

## 2.1 产品金字塔模型

```
           ┌─────────────┐
           │  超高价定制   │ $3,000+（1对1/企业咨询）
           │   5%客户     │
           ├─────────────┤
           │  高价VIP品   │ $999-$2,999（训练营/年度会员）
           │   15%客户    │
           ├─────────────┤
           │  中价核心品   │ $99-$999（系统课程）
           │   30%客户    │
           ├─────────────┤
           │  低价引流品   │ $9-$99（电子书/模板/迷你课）
           │   50%客户    │
           ├─────────────┤
           │   免费内容    │ 公域曝光 + 信任建立
           │   所有人      │
           └─────────────┘
```

## 2.2 各层级产品设计

### 免费内容

**目的：** 曝光 + 信任 + 引流

**形式：** 小红书笔记、公众号文章、短视频、知乎回答

**关键：** 每篇内容都要有明确的CTA（引导关注/加群/领资料）

### 低价引流品（$9-$99）

**目的：** 筛选付费用户 + 建立付费习惯

**形式：**
- 电子书（$9.9）
- 模板合集（$19.9）
- 迷你课程（$49）
- 工具包（$29.9）

**设计原则：**
- 低门槛，容易决策
- 高价值，超预期
- 能引导到中价产品

### 中价核心品（$99-$999）

**目的：** 核心利润来源

**形式：**
- 系统课程（$299-$499）
- 小班训练营（$599-$999）
- 付费社群年度会员（$199-$399）

**设计原则：**
- 解决一个完整问题
- 有明确的学习路径
- 能看到实际效果

### 高价VIP品（$999-$2,999）

**目的：** 深度服务 + 高利润

**形式：**
- 高端训练营（$1,999）
- 年度VIP会员（$999）
- 小组辅导（$1,499）

**设计原则：**
- 个性化服务
- 深度互动
- 明确的成果承诺

### 超高价定制品（$3,000+）

**目的：** 超级用户服务 + 品牌背书

**形式：**
- 1对1年度顾问（$5,000-$10,000）
- 企业咨询（$10,000+）
- 定制方案（按需定价）

## 2.3 产品组合策略

### 引流 → 转化路径

```
免费内容 → 低价引流品 → 中价核心品 → 高价VIP品
  100%       10-20%        3-5%         1-2%
```

### 关键指标

| 指标 | 健康值 |
|------|--------|
| 免费→低价转化率 | 10-20% |
| 低价→中价转化率 | 3-5% |
| 中价→高价转化率 | 1-2% |
| 客户终身价值 | >$500 |

## 2.4 产品迭代与淘汰

### 产品生命周期

```
新品期 → 成长期 → 成熟期 → 衰退期 → 淘汰/升级
```

### 迭代信号

- 转化率持续下降 → 需要更新内容
- 客户反馈差 → 需要改进质量
- 市场变化大 → 需要调整方向
- 竞品更好 → 需要差异化

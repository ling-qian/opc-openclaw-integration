## 6.1 复利效应的底层逻辑

```
第1个月：0 → $100（起步）
第3个月：$100 → $500（缓慢增长）
第6个月：$500 → $2,000（加速）
第12个月：$2,000 → $10,000（飞轮效应）
第24个月：$10,000 → $50,000+（指数增长）
```

**复利 = 内容积累 + 信任积累 + 客户积累 + 口碑积累**

## 6.3 构建你的飞轮

### 飞轮的六个齿轮

| 齿轮 | 作用 | 如何强化 |
|------|------|----------|
| 内容 | 吸引注意力 | 日更，提升质量 |
| 信任 | 建立专业度 | 案例，持续输出 |
| 产品 | 提供价值 | 迭代，超预期 |
| 客户 | 获得收入 | 服务，复购 |
| 口碑 | 扩大影响 | 激励分享 |
| 数据 | 优化决策 | 监控，复盘 |

### 飞轮启动策略

```
初期（飞轮还没转起来）：
→ 手动推（主动引流、私信、合作）
→ 用外力推（付费推广、KOL合作）

中期（飞轮开始转）：
→ 优化每个齿轮的效率
→ 减少手动，增加自动

后期（飞轮自转）：
→ 飞轮自己产生动力
→ 你只管加油和维护
```

## 6.4 长期主义心态

### 为什么大多数人失败

```
期望：1个月就能赚到钱
现实：3-6个月才有稳定收入

期望：发10条内容就爆
现实：可能要发100条

期望：1个产品就成功
现实：需要迭代3-5个版本
```

### 长期主义的核心

1. **做时间的朋友** — 每天进步1%，一年后完全不同
2. **不比较** — 每个人的起点和路径不同
3. **关注过程** — 结果是过程的副产品
4. **接受低谷** — 低谷是下一次爆发的积蓄

## 6.5 从副业到事业的跃迁

### 什么时候考虑全职

```
✅ 月收入 > 工资的100%，且持续3个月+
✅ 有6个月以上的生活费储备
✅ 有明确的增长路径
✅ 你真的热爱这件事
❌ 只是因为上班不开心
❌ 还没验证商业模式
❌ 没有安全垫
```

### 跃迁步骤

```
1. 兼职验证（3-6个月）
2. 收入稳定（3个月连续达标）
3. 储备安全垫（6个月生活费）
4. 辞职过渡（先兼职→再全职）
5. 全职加速（全力投入增长）
```

# SC3: 流量引擎 — 打造持续获客的流量系统

> 没有流量，一切都是零。构建你的流量飞轮

---

## 3.1 流量的本质

```
流量 = 被目标用户看到的机会

有机流量：靠内容吸引（免费，慢，稳）
付费流量：靠广告投放（付费，快，可控）
社交流量：靠关系裂变（免费，快，需要基础）
搜索流量：靠SEO优化（免费，慢，长尾）
```

## 3.2 有机流量系统

### 内容矩阵

| 内容类型 | 目的 | 频率 | 平台 |
|----------|------|------|------|
| 引流内容 | 曝光 + 关注 | 日更 | 小红书/抖音 |
| 信任内容 | 建立专业度 | 每周 | 公众号/知乎 |
| 转化内容 | 引导付费 | 按需 | 全平台 |
| 裂变内容 | 用户分享 | 每周 | 全平台 |

### 内容流量漏斗

```
曝光（10000次/月）
  ↓ 5% 关注
关注（500人/月）
  ↓ 10% 加群
入群（50人/月）
  ↓ 20% 付费
成交（10人/月）
```

## 3.3 社交流量裂变

### 裂变机制

```
用户A看到好内容 → 分享到朋友圈
  → 用户B看到 → 关注 → 也分享
    → 用户C、D、E看到...
      → 指数级增长
```

### 促进裂变的方法

1. **内容足够好** — 值得分享是前提
2. **降低分享门槛** — 提供现成的分享语/图
3. **激励分享** — 分享领资料/优惠
4. **社交货币** — 分享后显得用户有品位/专业

## 3.4 搜索流量（SEO）

### 关键词策略

```
核心词：一人公司
长尾词：一人公司怎么起步、一人公司60天计划、一人公司赚钱方法
```

### SEO优化要点

- 标题包含关键词
- 内容结构清晰（小标题、列表）
- 持续更新（搜索引擎喜欢活跃的账号）
- 互相引用（内链、外链）

## 3.5 付费流量（进阶）

### 什么时候用付费流量

```
✅ 已跑通转化模型
✅ 知道单客获取成本
✅ 产品利润率能覆盖广告成本
❌ 还没验证产品 → 先做有机流量
❌ 转化率太低 → 先优化转化
```

### 付费流量渠道

| 渠道 | 适合 | 成本 |
|------|------|------|
| 小红书薯条 | 种草类内容 | ¥50-500/篇 |
| 抖音DOU+ | 短视频推广 | ¥100-1000/条 |
| 公众号互选 | 精准投放 | ¥0.5-2/曝光 |
| 微信广告 | 私域引流 | ¥1-5/关注 |

## 3.6 流量监控指标

| 指标 | 健康值 | 说明 |
|------|--------|------|
| 日均曝光 | >1000 | 内容被看到的次数 |
| 关注转化率 | >3% | 看到→关注的比例 |
| 粉丝增长率 | >10%/月 | 粉丝增长速度 |
| 内容互动率 | >5% | 点赞+评论+收藏/曝光 |
| 私域转化率 | >10% | 粉丝→加微信的比例 |

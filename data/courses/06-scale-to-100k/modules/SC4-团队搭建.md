# SC4: 团队搭建 — 一人公司的"团队"架构

> 一个人≠什么都自己干。聪明地借力，才能规模化

---

## 4.1 一人公司的"团队"定义

```
你（核心决策 + 高价值工作）
  + 外包人员（执行层面的工作）
  + AI工具（重复性工作）
  + 协作者（互补技能的合作）
  = 虚拟团队
```

## 4.2 外包策略

### 什么该外包

| 工作类型 | 是否外包 | 原因 |
|----------|----------|------|
| 内容创作 | ❌ | 核心竞争力 |
| 战略决策 | ❌ | 只有你懂 |
| 视觉设计 | ✅ | 可标准化 |
| 视频剪辑 | ✅ | 耗时但可外包 |
| 客服回复 | ✅ | 可用模板+AI |
| 数据录入 | ✅ | 纯执行 |
| 财务记账 | ✅ | 专业性强 |

### 外包渠道

| 平台 | 适合 | 价格区间 |
|------|------|----------|
| 猪八戒 | 设计/开发/文案 | ¥100-5000 |
| 闲鱼 | 简单任务 | ¥10-500 |
| 大学生兼职 | 内容整理/数据 | ¥50-200/天 |
| 专业外包公司 | 系统性工作 | 按项目报价 |

### 外包管理要点

1. **写清楚SOP** — 标准操作流程
2. **给模板** — 不要让外包猜你想要什么
3. **设检查点** — 分阶段验收，不要最后才发现问题
4. **保留知识** — 外包走了，知识还在

## 4.3 AI作为团队成员

### AI能做什么

| 角色 | AI工具 | 工作内容 |
|------|--------|----------|
| 写作助手 | ChatGPT/文心一言 | 初稿、改写、翻译 |
| 设计助手 | Canva AI/MidJourney | 配图、海报、封面 |
| 数据分析师 | ChatGPT/Notion AI | 数据分析、趋势洞察 |
| 客服 | AI聊天机器人 | 自动回复常见问题 |
| 项目经理 | Notion AI | 任务管理、进度跟踪 |

### AI工作流示例

```
内容创作流程：
1. 你：确定选题 + 核心观点
2. AI：生成初稿
3. 你：修改 + 加入个人经验
4. AI：优化标题 + 排版
5. 你：最终审核 + 发布
```

## 4.5 从个人到团队的演进

```
阶段1：纯个人（$0-$1,000/月）
  → 你做所有事
  → 重点：验证模式

阶段2：AI+外包（$1,000-$5,000/月）
  → AI处理重复工作
  → 外包执行层面
  → 重点：提升效率

阶段3：小团队（$5,000-$20,000/月）
  → 1-2个兼职/全职助手
  → 你专注核心
  → 重点：放大规模

阶段4：公司化（$20,000+/月）
  → 正式团队
  → 系统化运营
  → 重点：品牌与生态
```

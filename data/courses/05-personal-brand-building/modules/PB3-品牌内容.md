# PB3: 品牌内容 — 用内容塑造品牌形象

> 每一条内容都在为你的品牌加分或减分

---

## 3.1 品牌内容的底层逻辑

内容不只是"发东西"，而是在用户心中建立 **固定的认知**：

```
看到你的名字 → 想到什么？
看到你的内容 → 什么感觉？
遇到相关问题 → 会不会想到你？
```

**品牌内容 = 持续输出 + 统一风格 + 明确价值**

## 3.2 品牌内容三要素

### 要素一：固定的内容支柱

选择 2-3 个核心话题，所有内容围绕这些话题展开：

| 内容支柱 | 占比 | 示例 |
|----------|------|------|
| 核心专业 | 60% | 一人公司方法论 |
| 个人故事 | 25% | 你的创业经历、踩坑故事 |
| 行业观察 | 15% | 趋势分析、热点评论 |

### 要素二：统一的表达风格

- **语言风格**：是专业严谨，还是轻松幽默？
- **视觉风格**：配色、排版、图片风格统一
- **互动风格**：回复评论的语气和方式

### 要素三：可识别的内容标签

让用户一看就知道"这是你的内容"：
- 固定的开头/结尾语
- 特有的排版格式
- 标志性的金句/口头禅

## 3.3 品牌内容创作框架

```
每周内容规划：

周一：专业干货（核心支柱）
周二：案例拆解（核心支柱）
周三：个人故事（故事支柱）
周四：互动话题（增强粘性）
周五：行业观察（观察支柱）
周六：长内容（深度建立信任）
周日：复盘 + 下周规划
```

## 3.4 品牌故事模板

### 你的创业故事

```
标题：为什么我开始做 [领域]

[起点]：我曾经也是 [目标人群的状态]

[痛点]：直到有一天 [触发事件]

[转变]：我开始 [做了什么改变]

[结果]：[具体成果]

[使命]：现在我帮助 [人群] 实现 [目标]
```

### 客户成功故事

```
标题：[客户名/匿名] 如何从 [起点] 到 [终点]

[困境]：[客户之前的痛点]

[尝试]：试过哪些方法，为什么没用

[转折]：如何找到你，开始什么服务

[过程]：服务中的关键突破

[结果]：具体数据/变化

[感悟]：客户怎么说
```

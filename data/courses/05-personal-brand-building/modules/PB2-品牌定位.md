# PB2: 品牌定位

> 找到你的独特价值主张

---

## 🎯 模块目标

找到你的独特价值主张，让你的品牌在众多创作者中脱颖而出。

**学完本模块，你将能够：**
- 找到你的独特价值主张（UVP）
- 明确你的目标受众
- 建立你的品牌差异化
- 用一句话说清楚你是谁

---

## 📖 核心知识点

### 1. 什么是独特价值主张（UVP）

UVP = 你能为用户提供什么独特的价值？

```
UVP = 你的技能 × 你的经验 × 你的个性 × 市场需求
```

**好的UVP：**
- 清晰：一句话说清楚
- 独特：和别人不一样
- 有价值：能帮用户解决问题

### 2. 找到UVP的3个方法

**方法1：技能+经验交叉**
```
你的技能：______
你的经验：______
交叉点：______
```

**方法2：问题+解决方案**
```
你最常被问什么问题？
你最擅长解决什么问题？
你的解决方案是什么？
```

**方法3：差异化定位**
```
同赛道的人都在做什么？
你能提供什么不同的？
你的独特角度是什么？
```

### 3. 品牌定位公式

```
我帮助 [目标人群] 通过 [你的方法] 实现 [他们想要的结果]
```

**例子：**
- 我帮助 **上班族** 通过 **写作** 实现 **副业月入5000**
- 我帮助 **小企业主** 通过 **短视频** 实现 **获客量翻倍**
- 我帮助 **自由职业者** 通过 **AI工具** 实现 **效率提升10倍**

### 4. 目标受众画像

| 维度 | 问题 |
|------|------|
| **基本特征** | 年龄、性别、职业、收入 |
| **痛点** | 他们最头疼的问题是什么？ |
| **渴望** | 他们最想要什么？ |
| **行为** | 他们在哪里获取信息？ |
| **语言** | 他们怎么描述自己的问题？ |

### 5. 品牌差异化的5个维度

| 维度 | 说明 | 例子 |
|------|------|------|
| **角度** | 你切入的独特角度 | 「从心理学角度讲营销」 |
| **方法** | 你的独特方法论 | 「3步写作法」 |
| **风格** | 你的表达风格 | 「幽默风趣讲知识」 |
| **人群** | 你服务的特定人群 | 「专为宝妈设计的副业课」 |
| **经历** | 你的独特经历 | 「从月薪3000到年入100万」 |

## ✅ 作业

1. **找到你的UVP**：用公式写出你的独特价值主张
2. **明确目标受众**：详细描述你的目标用户画像
3. **做竞品分析**：分析3个同赛道的创作者，找到你的差异化
4. **设计品牌视觉**：确定你的品牌颜色、字体、风格

---

## ⚠️ 常见误区

### 误区1：「我的定位太窄了」

**真相：** 定位越窄，越容易建立专业感。先聚焦一个细分领域，再扩展。

### 误区2：「我什么都能做」

**真相：** 什么都做 = 什么都做不好。先做好一件事，再扩展。

### 误区3：「定位不能改」

**真相：** 定位可以调整。先定一个，边做边优化。

### 误区4：「我要找到完美的定位」

**真相：** 没有完美的定位。先开始，再迭代。

### 误区5：「我和别人差不多」

**真相：** 每个人都有独特的经历、角度、风格。找到你的独特之处。

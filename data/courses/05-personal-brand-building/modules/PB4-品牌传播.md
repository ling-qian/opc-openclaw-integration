# PB4: 品牌传播 — 多平台联动放大影响力

> 一次创作，多平台分发，让品牌影响力指数级放大

---

## 4.1 多平台策略

### 平台角色定位

| 平台 | 角色 | 内容类型 | 目标 |
|------|------|----------|------|
| 公众号 | 大本营 | 深度长文 | 信任建立 |
| 小红书 | 前线阵地 | 图文/短视频 | 曝光引流 |
| 知乎 | 专业背书 | 问答/专栏 | SEO长尾 |
| 抖音 | 流量入口 | 短视频 | 快速曝光 |
| B站 | 深度连接 | 中长视频 | 粉丝粘性 |
| 视频号 | 私域入口 | 短视频/直播 | 微信生态联动 |

### 内容分发策略

```
核心内容（长文/视频）
  ↓ 改编
├── 公众号原文（2000字）
├── 小红书精简版（500字+图）
├── 知乎回答版（适配问答格式）
├── 抖音口播版（60秒精华）
├── B站深度版（10分钟解读）
└── 朋友圈金句版（1句话+图）
```

## 4.2 内容复用矩阵

一次深度思考，产出多个内容：

| 原始内容 | 改编方向 |
|----------|----------|
| 长文 | 金句卡片、信息图、短视频脚本 |
| 视频 | 文字稿、精华截图、播客音频 |
| 直播 | 精华剪辑、图文回顾、FAQ合集 |
| 案例 | 故事版、数据版、对比版 |

## 4.3 跨平台引流

```
小红书（公域曝光）
  ↓ 引导关注公众号
公众号（深度信任）
  ↓ 引导加微信/进群
微信社群（私域运营）
  ↓ 转化付费
付费产品/服务
```

**关键原则：**
- 公域做曝光，私域做转化
- 每个平台只做一件事，不要贪多
- 引流路径越短越好

## 4.4 品牌传播节奏

### 日常传播

```
每天：3条短内容（多平台同步）
每周：1篇长内容（公众号首发）
每月：1次直播/线上分享
每季：1个大型活动/课程发布
```

### 爆发传播

利用热点/节日/里程碑做集中传播：
- 行业热点 → 快速出观点
- 个人里程碑 → 故事分享
- 节日/季节 → 主题活动

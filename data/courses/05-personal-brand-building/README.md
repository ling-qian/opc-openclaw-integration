# 课程05：个人品牌打造课

## Personal Brand Building

> "Your brand is what people say about you when you're not in the room." — Jeff Bezos

---

## 🎯 课程目标

从内容创作者升级为个人品牌IP，建立长期竞争壁垒。

**学完这门课，你将能够：**
- 找到你的独特价值主张
- 用内容塑造有辨识度的品牌形象
- 通过多平台联动放大影响力
- 建立从品牌到收入的转化路径
- 打造持续增值的品牌护城河

---

## 📋 课程信息

| 项目 | 详情 |
|------|------|
| **适合人群** | 有内容基础，想从创作者升级为IP的进阶者 |
| **课时** | 6个模块，60天品牌塑造 |
| **每日投入** | 1-1.5小时 |
| **前置要求** | 已有持续的内容产出 |

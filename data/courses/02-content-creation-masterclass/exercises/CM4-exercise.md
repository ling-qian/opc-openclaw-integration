# CM4 实操作业：视频脚本写作练习

## 📋 作业目标

掌握短视频脚本的写作方法，完成3种不同类型的视频脚本，为实际拍摄做好准备。

## 🎯 具体任务

### 任务1：视频类型认知

了解3种常见短视频类型的特点：

| 类型 | 时长 | 特点 | 适用场景 |
|------|------|------|---------|
| 口播型 | 30-60秒 | 真人出镜+讲述 | 干货分享、观点输出 |
| 教程型 | 1-3分钟 | 屏幕录制+讲解 | 技能教学、工具演示 |
| 剧情型 | 30-90秒 | 情景演绎 | 故事营销、痛点展示 |

## ✅ 完成标准

- [ ] 口播脚本：30-60秒，有明确的开头钩子和结尾CTA
- [ ] 教程脚本：1-3分钟，步骤清晰，有画面描述
- [ ] 剧情脚本：30-90秒，有冲突-转折-解决的完整结构
- [ ] 每个脚本都有字幕/贴纸的设计
- [ ] 每个脚本都标注了时长分配

## 💡 参考范例

**口播脚本示例**：

```
【视频标题】：Excel里最被低估的一个函数

【开头钩子（3秒）】：
画面：对镜头，表情认真
台词："Excel里有一个函数，90%的人都不知道它有多强。"

【正文（35秒）】：
"它叫SUMPRODUCT。"
"看起来只是一个求和函数，但它能做的事情远比你想象的多。"
"比如，你想统计'销售部且业绩超过10万'的人数，"
"用COUNTIFS要写一大串，"
"但用SUMPRODUCT，一行公式搞定。"
（屏幕演示公式）

【结尾CTA（5秒）】：
"想知道怎么用？评论区告诉我，我出详细教程。"
字幕：评论区扣1，教你SUMPRODUCT高级用法
```

**剧情脚本示例**：

```
【视频标题】：领导说"这个PPT不行"的时候...

【场景1：问题呈现（15秒）】
画面：办公室，主角紧张地站在领导旁边
领导：（皱眉）"这个PPT...不太行啊。"
主角：（内心OS）"又来了，这已经是第3次了..."

【场景2：转折（20秒）】
画面：主角回到工位，打开电脑搜索
字幕："后来我发现，不是我PPT做得差..."
画面：打开一个模板网站
字幕："而是我不知道领导想看什么。"

【场景3：解决（20秒）】
画面：主角用新方法重新做PPT
字幕："用了这个方法后..."
画面：领导看PPT，点头

【场景4：结果（15秒）】
领导：（微笑）"这次不错。"
字幕："方法比努力更重要"
结尾字幕："想要这个方法？看我主页"
```

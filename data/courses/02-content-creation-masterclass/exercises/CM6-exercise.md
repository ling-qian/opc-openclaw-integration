## 💡 参考范例

**AI选题Prompt示例**：

```
你是一个Excel办公技巧领域的资深内容创作者，请帮我生成20个小红书选题。

我的目标用户：25-35岁的职场白领，经常需要做报表和汇报PPT。

要求：
1. 覆盖干货、故事、资源、互动4种类型
2. 每种类型至少3个
3. 标题要吸引人，包含数字或情绪词
4. 要有"职场场景"感，让打工人有共鸣
```

**AI工作流示例**：

| 步骤 | 我的做法 |
|------|---------|
| 选题 | 让AI生成20个→我选3个→问AI哪个最有潜力 |
| 标题 | 先自己写1个→让AI改5个→我选最好的 |
| 正文 | 让AI写初稿→我删掉废话→加真实案例→调整语气 |
| 配图 | 用Canva模板→AI生成背景→我调整文字和排版 |
| 发布 | 固定时间手动发布→用AI写评论区引导语 |

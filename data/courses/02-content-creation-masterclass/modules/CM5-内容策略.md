# CM5: 内容策略

> 选题规划、内容日历、数据复盘

---

## 🎯 模块目标

建立系统化的内容策略，让你的内容创作有规划、有节奏、有复盘。

**学完本模块，你将能够：**
- 建立可持续的选题库
- 制定并执行内容日历
- 用数据驱动内容优化
- 形成内容创作的正循环

---

## 📖 核心知识点

### 1. 选题的四个来源

| 来源 | 方法 | 例子 |
|------|------|------|
| **用户需求** | 收集常见问题 | 「怎么开始写作？」 |
| **行业热点** | 追踪行业动态 | 「ChatGPT对写作的影响」 |
| **个人经历** | 分享真实故事 | 「我做副业的第一个月」 |
| **竞品参考** | 分析对标账号 | 「这个博主为什么能火」 |

**选题金矿：你的评论区和私信。** 用户问的问题，就是最好的选题。

### 2. 选题矩阵

用「热度×相关度」矩阵筛选选题：

```
            高热度
              │
    ┌─────────┼─────────┐
    │ 优先做   │ 谨慎做   │
    │（高热高关）│（高热低关）│
低相关────────┼────────高相关
    │ 不做     │ 持续做   │
    │（低热低关）│（低热高关）│
    └─────────┼─────────┘
              │
            低热度
```

**原则：** 优先做「高热度+高相关度」的选题。

### 3. 内容日历的搭建

**内容日历 = 你的内容生产计划表**

**搭建步骤：**
1. 确定发布频率（建议：每周3-5条）
2. 分配内容类型（干货/故事/观点/互动）
3. 填入选题
4. 标注负责平台
5. 设定deadline

### 4. 数据复盘的核心指标

| 指标 | 含义 | 优化方向 |
|------|------|----------|
| **阅读量/播放量** | 内容曝光 | 优化标题和封面 |
| **点赞率** | 内容质量 | 提升内容价值 |
| **评论率** | 互动性 | 增加互动引导 |
| **收藏率** | 实用性 | 增加干货密度 |
| **转发率** | 传播性 | 增加共鸣点 |
| **关注转化率** | 吸引力 | 优化人设和定位 |

### 5. 内容复盘的5个问题

每篇内容发布后，问自己：
1. 这篇内容的数据表现如何？
2. 哪个指标最好/最差？为什么？
3. 用户的评论/反馈是什么？
4. 下次可以改进什么？
5. 这个选题值得继续深挖吗？

## 🛠️ 实操方法

### 方法1：建立选题库

**选题库分类：**
```
📁 选题库
├── 📁 用户问题（从评论/私信收集）
├── 📁 行业热点（追踪行业动态）
├── 📁 个人经历（你的真实故事）
├── 📁 竞品参考（对标账号的爆款）
├── 📁 系列选题（可以做成系列的）
└── 📁 临时灵感（随时记录的灵感）
```

**目标：** 选题库里永远有30个以上的备选选题。

### 方法2：制定周内容日历

| 日期 | 平台 | 内容类型 | 选题 | 状态 |
|------|------|----------|------|------|
| 周一 | 小红书 | 干货 | [选题] | 待写 |
| 周二 | 公众号 | 深度 | [选题] | 待写 |
| 周三 | 朋友圈 | 日常 | [选题] | 待写 |
| 周四 | 小红书 | 故事 | [选题] | 待写 |
| 周五 | 知乎 | 问答 | [选题] | 待写 |
| 周六 | - | 休息 | - | - |
| 周日 | - | 储备 | - | - |

### 方法3：每周数据复盘

**每周日晚上，花30分钟做复盘：**

1. **统计本周数据**
   - 发了多少条内容？
   - 总阅读/播放量多少？
   - 最高/最低的是哪条？

2. **分析原因**
   - 最高：为什么好？（选题？标题？时间？）
   - 最低：为什么差？（可以怎么改进？）

3. **制定下周计划**
   - 保留好的做法
   - 改进不足的地方
   - 尝试新的方法

## ✅ 作业

1. **建立选题库**：收集30个以上的备选选题，分类整理
2. **制定下周内容日历**：规划下周的每一天的内容
3. **做一次数据复盘**：如果你已经有内容产出，做一次完整的数据复盘
4. **优化3篇旧内容**：找3篇表现不好的旧内容，重新优化标题和内容

## ⚠️ 常见误区

### 误区1：「想到什么发什么」

**真相：** 随机发布 = 随机结果。有计划的内容创作，效率和效果都更好。

### 误区2：「我不需要看数据」

**真相：** 数据是你的指南针。不看数据，你永远不知道什么有效、什么无效。

### 误区3：「内容日历太死板了」

**真相：** 内容日历是指南，不是枷锁。可以灵活调整，但不能没有。

### 误区4：「我只要做好内容就行」

**真相：** 内容只是起点。选题、策略、复盘，才是让你持续进步的关键。

### 误区5：「对标就是抄袭」

**真相：** 对标是学习成功者的经验。看他们做对了什么，你能学到什么。不是抄内容，是学方法。

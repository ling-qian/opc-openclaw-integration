# CM2: 短内容创作

> 300字以内的爆发力（小红书/抖音/朋友圈）

---

## 🎯 模块目标

掌握短内容的创作方法论，让你每天轻松产出高质量的短内容。

**学完本模块，你将能够：**
- 写出高互动的小红书笔记
- 制作有传播力的抖音文案
- 打造有吸引力的朋友圈内容
- 建立短内容创作的肌肉记忆

---

## 📖 核心知识点

### 1. 短内容的核心法则：3秒定生死

短内容的战场，只有3秒。用户刷到你的内容，3秒内决定看不看。

**3秒法则：**
- **标题/封面**：必须在3秒内抓住注意力
- **第一句话**：必须让人想读第二句
- **视觉呈现**：干净、清晰、有重点

### 2. 短内容的四种类型

| 类型 | 适用场景 | 例子 |
|------|----------|------|
| **干货型** | 分享方法、技巧、工具 | 「3个方法让你的PPT颜值翻倍」|
| **故事型** | 分享经历、案例、转折 | 「从月薪3000到年入50万，我做了这3件事」|
| **观点型** | 表达看法、引发讨论 | 「为什么我不建议你考公」|
| **情绪型** | 引发共鸣、戳中痛点 | 「30岁了还一事无成，正常吗？」|

### 3. 平台差异与适配

| 平台 | 内容特点 | 最佳长度 | 关键指标 |
|------|----------|----------|----------|
| **小红书** | 图文并茂、实用干货 | 300-500字 | 收藏率 |
| **抖音** | 短视频为主、文案配合 | 15-60秒 | 完播率 |
| **朋友圈** | 真实、日常、有温度 | 50-200字 | 点赞数 |
| **微博** | 观点、热点、互动 | 140字以内 | 转发数 |

### 4. 爆款标题公式

**公式1：数字 + 结果**
- 「5个方法让你的公众号阅读量翻倍」
- 「30天减20斤，我只做了这一件事」

**公式2：痛点 + 解决方案**
- 「总是写不出东西？因为你没用这个框架」
- 「不知道发什么朋友圈？这10个模板够你用一年」

**公式3：反常识 + 新认知**
- 「为什么越努力越穷」
- 「我劝你别太自律」

**公式4：人群 + 场景**
- 「上班族必看：下班后2小时如何高效利用」
- 「创业者最常犯的5个错误」

### 5. 短内容的结构框架

**AIDA框架（最经典）：**
```
A - Attention：抓住注意力（标题/开头）
I - Interest：引发兴趣（痛点/场景）
D - Desire：激发欲望（好处/案例）
A - Action：促进行动（CTA）
```

**PAS框架（适合痛点型）：**
```
P - Problem：描述问题
A - Agitate：放大痛点
S - Solution：给出方案
```

## 🛠️ 实操方法

### 方法1：每日3条朋友圈训练

每天发3条朋友圈，分别练习：
1. **干货型**：分享一个你学到的知识点
2. **故事型**：分享一个你今天的经历
3. **观点型**：分享一个你对某件事的看法

### 方法2：小红书笔记创作流程

```
Step 1：选题（从选题库中选）
Step 2：写标题（用爆款公式，写5个标题选1个）
Step 3：写正文（用AIDA框架，300-500字）
Step 4：配图（Canva制作封面 + 3-5张配图）
Step 5：检查（用价值评估框架自检）
Step 6：发布（选择最佳发布时间）
```

### 方法3：10分钟速写训练

设定10分钟倒计时，随便选一个话题，不停笔地写。写多少算多少，不修改，不回头。

**目的：** 打破"不知道写什么"的心理障碍，训练写作肌肉。

## 📝 模板/框架

### 小红书笔记模板

```
【标题】（用爆款公式）

📌 核心观点（一句话）

1️⃣ 第一个要点
- 具体说明
- 举例/数据

2️⃣ 第二个要点
- 具体说明
- 举例/数据

3️⃣ 第三个要点
- 具体说明
- 举例/数据

💡 总结金句

#标签1 #标签2 #标签3
```

### 朋友圈文案模板

**模板1：日常分享**
```
今天做了______，发现______。
总结一句话：______。
#日常 #成长
```

**模板2：干货分享**
```
最近学到一个很有用的______方法：
1. ______
2. ______
3. ______
试试看，效果真的不错👍
```

**模板3：故事分享**
```
今天遇到一件事，______。
让我想起______。
最后我的感悟是______。
```

## ✅ 作业

1. **连续7天发朋友圈**：每天至少发1条，练习不同类型的短内容
2. **写5条小红书笔记**：用模板创作5条笔记，发到小红书（可以先存草稿）
3. **爆款标题练习**：针对你的赛道，用4个公式各写5个标题
4. **10分钟速写**：连续3天，每天做一次10分钟速写训练

## ⚠️ 常见误区

### 误区1：「短内容不需要深度」

**真相：** 短内容不是没深度，而是把深度浓缩成精华。300字能说清楚的事，不需要3000字。

### 误区2：「每个平台发一样的内容」

**真相：** 每个平台的用户习惯不同，内容形式也要适配。小红书重图文，抖音重视频，朋友圈重真实。

### 误区3：「标题党就是好标题」

**真相：** 好标题是吸引人点击，同时内容要配得上标题。标题党只会消耗信任。

### 误区4：「发得越多越好」

**真相：** 质量 > 数量。一天发10条烂内容，不如发1条好内容。

### 误区5：「模仿别人就等于抄袭」

**真相：** 模仿框架和思路是学习，抄袭文字是抄袭。先模仿，再超越。
